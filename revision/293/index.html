버전 2.9.3 (빌드 번호 2.9.3.26)

릴리즈|2023-06-26

신규 기능 및 개선​

  1. 리포트 생성 기능 추가

  2. 카드 프린터 기능 추가

    • cardPresso 프로그램을 사용하여 BioStar 2에서 직접 카드를 프린트할 수 있도록 지원

  3. 시스템 백업 및 복원 기능 추가

    • 백업된 파일을 사용하여 시스템을 복원 할 수 있는 BioStar Restore 프로그램 제공(BioStar 2 설치 프로그램에 포함)

  4. BioStation 3 QR 코드 인증 지원

    • 펌웨어 v1.1.0 이상

  5. 키패드 백라이트 설정 기능 지원

    • XPass D2 (XPD2-GKDB) 하드웨어 V02M, 펌웨어 v1.7.1 이상

  6. 보안 취약점 개선

    • Moment.js 라이브러리 버전 업데이트

    • Lodash 라이브러리 버전 업데이트

    • enckey 파일 생성 로직 개선

  7. 개인 정보 DB 암호화를 사용할 때 CSV 불러오기 시 업데이트 속도 개선

  8. CSV 불러오기 시 CSV 파일 내에 UTF-16 유니코드를 사용하는 문자도 정상적으로 인식하도록 개선

  9. 근태 관리 보안 취약점 개선

    • Swagger UI 버전 업그레이드

    • X-Frame-Options 추가

    • iframe 취약점 개선(Unified Gateway)

  10. 근태 관리 메뉴에 사용자 추가 시 라이선스 종류에 따라 정해져 있는 최대 사용자 수가 초과된 경우 표시되는 팝업 메시지 개선

  11. 사용자 자동 동기화 → 특정 장치(해당하는 출입 그룹에 속한 장치만 동기화) 모드 사용 시 성능 개선

    • 장치 설정 속도 개선

    • 구역 활성/비활성 시 처리 속도 개선

    • 사용자 그룹 이동 시 동기화 속도 개선

  12. C서버에 쿼리 실행 시 불필요한 데이터베이스 체크 로직 개선

  13. 임시 테이블에 누락된 인덱스를 추가하여 성능 개선

  14. User API을 사용하여 임시 테이블 업데이트 또는 삭제 시 서브쿼리 동작 최적화

  15. 모니터링의 출입문 상태 조회 속도 개선

  16. 모바일 출입카드 등록 시 사용자 이름에 미지원 문자 사용 여부를 체크하도록 개선

  17. IP 인터폰 설정 메뉴의 특정 항목에 일부 기호 + - @ . 입력 지원

    • 사용자명, 인증ID, 내선 번호

  18. QR코드 인증 관련 이벤트 코드 추가

  19. 라이선스 메뉴 신설로 인한 위치 혼동을 방지하기 위해 기존 라이선스 메뉴 위치(설정 → 서버)에 안내 메시지 추가

  20. 사용자 정보에 이메일 항목이 입력되지 않은 상태에서 모바일 출입카드 등록 시 나타나는 오류 팝업 메시지 개선

  21. 장치 목록의 장치별 아이콘 시인성 개선

  22. 로그인 페이지 변경

버그 수정​

  1. 모바일 출입카드 등록 시 간헐적으로 등록 버튼이 비활성화 되는 문제 발생 버전: v2.8.9

  2. C서버에서 getTrackingInfo로 추적 데이터를 조회하지 못한 경우 서버가 충돌하는 문제 발생 버전: v2.6.0

  3. 특정 조건으로 검색된 사용자를 모두 선택 후 장치에서 삭제 진행 시 선택하지 않았던 사용자들까지 모두 삭제되는 문제 발생 버전: v2.7.1

  4. 서버 메뉴의 등록용 장치 설정에 추가된 장치를 장치 메뉴에서 삭제하여도 등록용 장치 설정 목록에서 삭제되지 않는 문제 발생 버전: v2.2.1

  5. 서버 메뉴의 등록용 장치 설정에 장치를 추가할 때 장치가 검색되지 않는 문제 발생 버전: v2.2.1

  6. 서버 메뉴의 세션 만료 시간이 설정된 상태에서 서버를 재시작하면 세션 만료 시간이 기본값(60분)으로 동작하는 문제 발생 버전: v2.8.14

  7. 대시보드의 사용 현황에 최대값을 초과한 항목의 값이 잘못 표시되는 문제 발생 버전: v2.0.0

  8. ID에 특정 알파벳 또는 기호가 포함되어 있는 사용자에게 발급된 액세스 온 카드가 사용 정지 처리된 이후에도 인증에 성공하는 문제 발생 버전: v2.7.14

  9. 사용자 메뉴에서 데이터 파일 가져오기 완료 후 장치에 동기화되지 않는 문제 발생 버전: v2.1.0

  10. 근태 관리 메뉴의 근무 스케줄에 부재가 설정되어 있을 때 승인자인 사용자가 삭제되면 근태 기록이 조회되지 않는 문제 발생 버전: v2.7.0

  11. 일부 1:N 인증 성공 이벤트가 근태에서 누락되는 문제 발생 버전: v2.8.6

  12. 버전을 2.8.8 이하에서 2.8.9 이상으로 업그레이드한 후 TA 설정에서 장치 추가 시 오류가 발생하는 문제 발생 버전: v2.8.9
