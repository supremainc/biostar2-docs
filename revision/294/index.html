버전 2.9.4 (빌드 번호 2.9.4.29)

릴리즈|2023-09-20

신규 기능 및 개선​

  1. 신규 장치 연결 지원

    • BioStation 2a

  2. 템플릿 온 모바일 기능 추가

    • BioStation 3 펌웨어 1.2.0 이상

  3. CoreStation에 무선 도어락 연결 지원

    • 지원되는 OSDP 안테나와 무선 도어락은 다음과 같습니다.

      • U&Z OSDP 안테나: CX8936

      • U&Z 무선 도어락: CX217x(핸들), CX212x(노브)

  4. 커스텀 스마트 카드 레이아웃 지원

    • XPass D2 펌웨어 1.7.1 이상

  5. 스마트 카드 레이아웃 설정 개선

  6. 장치 상세 페이지의 관리자 메뉴 UI 개선

  7. 액티브 디렉토리 메뉴의 BioStar 2 로그인에 사용 설정 메뉴 UX 개선

  8. OpenSSL 버전 업그레이드

  9. OpenJDK 업데이트

  10. SQL Injection 보안 취약점 개선

  11. 액티브 디렉토리 계정 설정 보안 취약점 개선

  12. 액티브 디렉토리 로그인과 BioStar 2 로그인에 사용되는 ID 저장소 분리

  13. 비디오 패키지 내 보안 취약점 개선

  14. 사용자가 잘못된 비밀번호로 로그인을 시도하여 설정된 비밀번호 최대 실패 허용 횟수를 초과했을 경우 감사 추적이 기록되도록 개선

  15. 사용자의 로그인 ID를 최초 관리자의 기본값 ID인 1로 수정할 수 없도록 개선

    • BioStar 2 v2.9.4로 업그레이드 후에는 로그인 ID가 1로 설정된 관리자 계정으로는 더 이상 접속할 수 없습니다.

  16. 사용자 그룹 동기화 성능 개선

  17. 사용자 자동 동기화 - 특정 장치(해당하는 출입 그룹에 속한 장치만 동기화) 모드 사용 시 성능 개선

    • 출입 그룹 수정 속도 개선

    • 장치 일괄 편집 시 속도 개선

    • 출입 등급, 층 등급 생성 속도 개선

    • 출입문 수정 및 삭제 속도 개선

  18. 장치와 서버가 재연결될 때 동기화 성능 개선

  19. 방문자 메뉴에서 사용자 검색 속도 개선

  20. 보안 크리덴셜 카드 발급 시 카드 ID에 0으로 시작하는 숫자가 입력된 경우 발급되지 않도록 개선

  21. 장치에 설정된 스마트 카드 레이아웃이 BioStar 2에 없는 경우 장치 초기화 안내 메시지가 표시되지 않도록 개선

  22. 장치에서 5회 이상 비정상적인 이벤트 로그를 서버에 전달하게되면 서버에서 더 이상 로그를 요청하지 않도록 개선

  23. 개인 정보 DB 암호화 기능을 사용 안 함으로 설정하여 복호화 진행 시 중복된 사용자 ID 또는 로그인 ID를 확인하는 과정 추가

  24. 포트 메뉴에서 지원하지 않는 포트 삭제

    • 리포트 클라우드 포트

  25. 사용자 추가 및 수정 시 사용자 이름에 지원하지 않는 문자가 포함되어 있는 경우 나타나는 오류 팝업 메시지에 지원하는 특수문자를 표시하여 알기쉽도록 개선

  26. Airfob Portal에서 휴면 상태인 계정이나 추가 인증 모드를 사용 중인 계정을 BioStar 2에서 모바일 출입카드에 연결할 경우 표시되는 오류 메시지 개선

  27. 모바일 출입 카드 메뉴에서 연결이 해제된 장치를 삭제하려고 할 경우 표시되는 오류 메시지 개선

  28. 일부 장치 이름 표기 변경

    • Xpass 2 → XPass 2

    • Xpass D2 → XPass D2

  29. 비주얼 페이스 모바일 등록 링크 메일의 일본어 문구 개선

버그 수정​

  1. 모니터링 메뉴의 일부 목록에서 PDF 인쇄 시 서버가 멈추는 문제 발생 버전: 2.9.3

  2. 사용자 메뉴에서 BioStar 2 QR을 발급한 후 저장하여도 이메일이 오지 않는 문제 발생 버전: 2.8.11

  3. 특정 장치 연결 시, 설정해놓은 근태 키가 간헐적으로 이벤트 로그에 표시되지 않는 문제 발생 버전: 2.8.14

  4. 출입 통제 메뉴에서 출입 그룹을 검색하고 목록에 나타난 출입 그룹의 상세 화면으로 진입하여 페이지 이동 버튼을 눌렀을 때 검색된 목록 내에서 이동되지 않고 전체 출입 그룹 목록에서 이동되는 문제 발생 버전: 2.3.0

  5. 등록된 그래픽 맵에서 갱신된 출입문 릴레이 상태를 불러오지 않는 문제 발생 버전: 2.8.16

  6. API를 이용하여 사용자 생성 시 다른 사용자가 사용 중인 카드를 할당하면 카드 중복되어 사용자 생성에 실패하는 문제 발생 버전: 2.8.14

  7. 화재 경보 구역을 추가할 때 모드를 반복적으로 변경하면 정상적으로 표시되지 않는 문제 발생 버전: 2.6.3

  8. 커스텀 권한 관리자로 접속했을 때 실시간 로그에서 필터링이 설정한대로 동작하지 않는 문제 발생 버전: 2.6.3

  9. BioStar 2에서 로그아웃하면 로그인이 필요합니다. 팝업이 표시되는 문제 발생 버전: 2.8.10

  10. 시스템 보안 메뉴에서 비밀번호 변경 주기를 활성화하면 액티브 디렉토리 계정으로 로그인이 되지않는 문제 발생 버전: 2.8.16

  11. CSV 불러오기를 이용하여 1000명 이상의 사용자를 생성하면 간헐적으로 실패하는 문제 발생 버전: 2.9.3

  12. CSV 불러오기를 사용하여 잘못된 사용자 ID 생성이 가능한 문제 발생 버전: 2.8.16

  13. AoC(Access on Card)가 발급된 사용자에 CSV 불러오기로 사용자 정보를 덮어쓰기하여 변경한 경우 AoC 정보가 삭제되지만 블랙리스트에 추가되지 않는 문제 발생 버전: 2.3.0

  14. 마스터-슬레이브-Wiegand 리더로 연결된 상태에서 특정 장치 그룹의 권한을 가진 사용자가 이벤트 로그 조회 시 Wiegand 리더의 동작 로그가 조회되지 않는 문제 발생 버전: 2.6.3

  15. 일부 언어의 리소스 파일을 추가하고 언어 설정을 해당 언어로 변경해도 해당 언어로 표시되지 않는 문제 발생 버전: 2.8.14

  16. BioEntry W2 장치 추가 시 간헐적으로 로그아웃 되는 문제 발생 버전: 2.9.0

  17. MSSQL을 사용하는 환경에서 1000대 이상의 장치를 전체 삭제할 경우 삭제되지 않는 문제 발생 버전: 2.9.3

  18. MSSQL을 사용하는 환경에서 사용자 검색이 되지 않는 문제 발생 버전: 2.8.0

  19. 이벤트 검색 API 호출 시 필수 파라미터를 채우지 않고 호출 했을 때 로그아웃되는 문제 발생 버전: 2.0.0

  20. 서버 매칭을 사용하는 경우 CSN Mobile 카드로 인증이 실패하는 문제 발생 버전: 2.9.0

  21. AoC(Access on Card) 스마트 카드 쓰기 실패 시 블랙리스트로 추가되는 문제 발생 버전: 2.4.1

  22. 암복호화 실행 시 서버가 정상적으로 종료되지 않는 문제 발생 버전: 2.8.0

  23. 액티브 디렉토리 메뉴의 BioStar 2 로그인에 사용이 활성화된 상태에서 사용자의 권한 등급을 없음으로 설정하면 로그인 ID 정보가 삭제되는 문제 발생 버전: 2.8.16

  24. 액티브 디렉토리 메뉴의 BioStar 2 로그인에 사용이 활성화된 상태에서 숫자로 시작되는 사용자 ID로 로그인하면 실패하는 문제 발생 버전: 2.8.16

  25. FaceStation F2 펌웨어 2.x.x 버전에서 사용자를 USB에 내보내기 한 후 BioStar 2에 데이터 파일 가져오기 진행 시 비주얼 페이스가 업로드되지 않는 문제 발생 버전: 2.8.10

    알아두기

    FaceStation F2 펌웨어 1.x.x 버전에서는 BioStar 2에서 더 이상 데이터 파일 내보내기를 지원하지 않습니다. 데이터 파일 내보내기를 사용하려면 장치 펌웨어를 업그레이드하세요.

  26. 모바일 출입 카드 설정을 사용 안 함으로 설정한 후 모바일 카드가 등록된 사용자를 삭제할 경우 불필요한 오류 팝업이 표시되는 문제 발생 버전: 2.9.0

  27. /api/users/data_export?id=*를 사용하여 API 호출 시 id에 입력된 사용자가 서버에 존재하지 않는 경우 서버가 종료되는 문제 발생 버전: 2.8.10

  28. API로 CSN 카드 발급 시 유효하지 않은 CARD_ID로 발급이 가능한 문제 발생 버전: 2.0.0

    • CARD_ID가 숫자 외의 값을 포함하는 경우

    • CARD_ID가 32자를 초과하는 경우

    • CARD_ID가 0으로 시작하는 경우

  29. DE-620과 카드의 기본 키를 사용하여 스마트 카드를 발급할 때 카드에 잘못된 키가 쓰여지는 문제 발생 버전: 2.8.14

  30. 비디오를 사용하는 환경에서 BioStar 2를 업그레이드한 후 이벤트 로그에 비디오 로그가 보이지 않는 문제 발생 버전: 2.8.4

  31. BioStar Setting에서 Test Connection 버튼을 클릭하면 BioStar Setting이 종료되는 문제 발생 버전: 2.9.3

  32. 리포트 메뉴에서 사용자 상세 정보 리포트 생성 시 사용자 그룹이 선택되어있는 사용자가 있을 경우 오류가 발생하는 문제 발생 버전: 2.9.3

  33. BioStar 2를 D드라이브에 설치하면 리포트 서비스가 시작되지 않는 문제 발생 버전: 2.9.3

  34. 모바일 카드로 인증 시 이벤트 로그에 소집 구역의 인증 이벤트가 업데이트되지 않는 문제 발생 버전: 2.8.6

  35. 장치를 선택하여 장치 상세페이지로 진입 시 간헐적으로 서버가 종료되는 문제 발생 버전: 2.0.0
