BioStar 2 업그레이드 시 주의사항

  • BioStar 2.9.2 버전부터는 32비트 운영체제용 설치 파일이 제공되지 않습니다.

  • BioStar 1이 설치된 PC에 BioStar 2를 설치하지 마세요. 프로그램 성능에 문제가 발생할 수 있습니다.

  • BioStar 2.2.1이나 2.2.2 버전이 설치된 상태에서 BioStar 2.3.0을 설치할 경우 SQLite에 저장된 모든 정보를 새로운 MariaDB로 마이그레이션합니다.

  • BioStar 2 최신 버전으로의 업그레이드는 버전 2.6.0에서부터 지원됩니다. BioStar 2.6.0 이하 버전을 사용 중인 경우, 최신 버전으로 업그레이드하려면 아래와 같이 2.6.0 이하 버전의 BioStar 2를 모두 순차적으로 설치하세요.

  • Windows 8.1을 사용하는 경우 KB2919355 업데이트를 설치하세요.

  • MS SQL 2012 Express를 사용할 경우 서비스 팩 3를 설치하세요.

  • MS SQL 2014 Express를 사용할 경우 서비스 팩 2를 설치하세요.

  • BioStar 2.9.9 버전부터는 MS SQL Server 2022 연결을 지원하며, 연결 방식으로 Microsoft OLE DB Driver for SQL Server 또는 Native Client를 사용할 수 있습니다.

    • OLE DB 드라이버가 설치된 환경에서는 OLE DB를 기본 연결 방식으로 사용하며, Native Client만 설치된 경우에는 Native Client로 연결됩니다.

    • Native Client는 SQL Server 2022부터 기본 설치되지 않으며, 호환성과 기능 측면에서 제한이 있으므로 최신 SQL Server와의 연결에는 OLE DB 드라이버 사용을 권장합니다.

  • MS SQL Server와 BioStar 2가 서로 다른 PC에 설치된 경우, BioStar 2가 설치된 PC에는 Microsoft OLE DB Driver for SQL Server 또는 Native Client를 설치해야 합니다.

    • TA 메뉴 사용 시에는 Microsoft ODBC Driver 17 for SQL Server가 필요하므로 환경에 맞게 확인하여 설치하세요.

  • MS SQL Server를 사용할 경우 서버 및 각 테이블의 Collation 옵션을 CI(Case-insensitive)로 설정하세요.

  • 기존 버전의 BioStar 2에서 데이터베이스를 백업할 때 모든 서비스를 사용 중지한 뒤 진행하세요. 또한, AC 데이터베이스와 TA 데이터베이스를 함께 백업하고 복원하지 않으면 TA 데이터베이스를 사용할 수 없게 됩니다.

  • BioStar 2 데이터베이스를 백업하려면 \Program Files\BioStar 2(x64)\util 폴더의 enckey 파일과 \Program Files\BioStar 2(x64) 폴더의 system.conf, setting.conf 파일도 반드시 함께 백업하세요. 그렇지 않으면, 데이터베이스를 사용할 수 없게 됩니다.

안내사항​

관리자 가이드 변경점 안내

BioStar 2.9.5 버전부터는 신기능이나 개선된 기능의 사용 방법에 대해 주제 단위로 분리하여 기능 가이드를 제공합니다.

FaceStation F2 v1.x.x 펌웨어 지원 중단 안내

BioStar 2 v2.9.7부터 사용자의 비주얼 페이스를 장치에 동기화할 때 템플릿만 전송하도록 변경하여 동기화 성능을 최적화했습니다.

따라서 BioStar 2 v2.9.7부터는 장치에서만 템플릿을 추출할 수 있는 FaceStation F2 v1.x.x 펌웨어의 지원이 중단됩니다.

현재 FaceStation F2의 1.x.x 버전 펌웨어를 사용 중인 경우 BioStar 2 v2.9.7을 사용하기 위해서는 FaceStation F2를 최신 펌웨어로 업그레이드하여 사용하세요.

최신 펌웨어는 슈프리마 다운로드 센터에서 다운로드할 수 있습니다.

특정 옵션에서 드롭다운 목록을 제대로 선택할 수 없는 경우 문제 해결 방법 (2.9.7 버전 이하)

크롬(Chrome)의 Keyboard focusable scrollers 기능이 활성화되어 있는 경우, 드롭다운 목록이 길어서 스크롤이 필요한 옵션을 선택할 때 키보드 포커스가 의도하지 않은 방식으로 동작하여, 사용자가 원하는 옵션을 제대로 선택할 수 없는 문제가 발생할 수 있습니다.

  • 해결 방법: 크롬(Chrome) 설정에서 Keyboard focusable scrollers 기능 비활성화

  1. 주소창에 chrome://flags/ 를 입력하세요.

  2. Keyboard focusable scrollers를 검색하세요.

  3. Keyboard focusable scrollers의 드롭다운 메뉴에서 Disabled를 선택하세요.

  4. 다시 시작을 눌러 크롬(Chrome)을 다시 시작하면 변경 사항이 적용됩니다.
