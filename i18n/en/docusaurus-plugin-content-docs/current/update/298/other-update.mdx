* Discontinuation of support for the BioStar 2 <Cmd sid="common.gnb.video" />.

* Improved <Cmd sid="common.gnb.door" /> menu UI.

    * Improved door <Cmd sid="door.openTime" /> setting UI.

    * Improved UI in the <Cmd sid="door.alarm" /> settings to differentiate between <Cmd sid="door.antiPassBack" /> and <Cmd sid="timedAPB.title" />.

* Improved <Cmd sid="common.gnb.report" /> performance.

* Improved <Cmd sid="common.dashboard" /> performance.

* Improved compatibility of user lists and <Cmd sid="button.advancedSearch" />.

* Added <Cmd sid="device.info.firmwareVersion" /> column to the device list in <Cmd sid="common.gnb.device" /> menu.

* Added <Cmd sid="common.elevator" /> and <Cmd sid="common.zone" /> columns to the <Cmd sid="columnSetting.columnList" /> of the <Cmd sid="monitoring.pastEvent" />.

* Added new predefined Wiegand card formats.

    * MIFARE CSN 32bit

    * MIFARE CSN 34bit (Parity)

    * DESFire 56bit

    * DESFire 58bit (Parity)
