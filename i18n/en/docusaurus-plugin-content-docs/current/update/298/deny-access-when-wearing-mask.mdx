Enhance facial authentication accuracy by rejecting authentication when a user is wearing a mask.

1. Click <Cmd sid="common.gnb.device" />.

1. Select the desired device from the device list.

    :::info

    The <Cmd>Deny access when wearing mask</Cmd> option is only available on BioStation 3 firmware version 1.3.1 or higher.

    :::

1. In the <Cmd sid="device.info.mask" /> settings, set <Cmd sid="device.mask.useMask" /> to <Cmd sid='device.mask.checkModeDenyMask' />.

    <Image src='/img/biostar-mask-detection-unmask.png' className='none' caption />

    :::info

    When <Cmd sid="device.mask.useMask" /> is set to <Cmd sid="device.mask.checkModeDenyMask" />, the <Cmd sid="device.thermal.maskOnlyMode" /> is fixed to <Cmd sid="device.thermal.authAfterMode" /> and cannot be changed.

    :::

1. Click <Cmd sid="button.apply" /> to save the settings.
