Suprema devices can display private messages on the screen during user authentication. Previously, this feature could only be set using the BioStar 2 Device SDK. However, starting with BioStar 2 v2.9.8, a private message option has been added to the server, allowing easier configuration of private messages directly from the server.

## How to Enter a Private Message in User Information

1. Click <Cmd sid="common.gnb.user" />.

1. Select the user for whom a private message will be set.

1. In the <Cmd sid="common.advanced" /> section, enter the message to be displayed during authentication in the <Cmd sid="common.privateMessage" /> field.

    <Image src='/img/en/biostar-298-private-message-1.png' className='none' caption />

1. Click <Cmd sid="button.apply" /> to save the settings.

## How to Enable the Server Private Message Option on a Device

1. Click <Cmd sid="common.gnb.device" />.

1. Select the device to set it from the device list.

1. In the <Cmd sid="device.info.advanced" /> section, go to the <Cmd sid="device.info.displaySound" /> menu and enable the <Cmd sid="device.info.serverPrivateMsg" /> option.

    <Image src='/img/en/biostar-private-message-2.png' caption />

1. Click <Cmd sid="button.apply" /> to save the settings.
