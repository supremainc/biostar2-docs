The existing search feature of the <Cmd sid="common.gnb.user" /> menu can only search for users by name and email, but the newly added <Cmd sid="button.advancedSearch" /> allows you to narrow the scope of a search query to eliminate irrelevant information to help you find the exact content you're looking for.

## How to Use Advanced Search

1. Click <Cmd sid="common.gnb.user" />.

1. Click <Cmd sid="button.advancedSearch" />.

    <Image src='/img/biostar-user-menu.png' caption />

    <Image src='/img/biostar-295-user-adv-search.png' className='none' caption />

    <div className='overflow-x'>

    | 항목 | 설명 |
    | -- | -- |
    | <Cmd sid="common.id" /> | Search using keywords included in the user's ID. |
    | <Cmd sid="common.name" /> | Search using keywords included in the user's name.<br/>* <Cmd sid="common.name" /> searches are case-insensitive. |
    | <Cmd sid="common.email" /> | Search using keywords included in the user's email. |
    | <Cmd sid="common.department" /> | Search using keywords included in the user's department.<br/>* If <Cmd sid="setting.sever.secure.comm.encrypt.db" /> is enabled, it can only be searched in full text. |
    | <Cmd sid="common.title" /> | Search using keywords included in the user's title.<br/>* If <Cmd sid="setting.sever.secure.comm.encrypt.db" /> is enabled, it can only be searched in full text. |
    | <Cmd sid="common.telephone" /> | Search using keywords included in the user's phone number. |
    | <Cmd sid="common.group" /> | Search for users in the selected group. |
    | <Cmd sid="common.accessGroup" /> | Search for users in the selected access group. |
    | <Cmd sid="common.status" /> | Search for users with <Cmd sid="user.status.active" />/<Cmd sid="user.status.inactive" /> status. |
    | <Cmd sid="user.bioStarOperator" /> | Search for users with the selected Operator Level. |
    | <Cmd sid="common.card" /> | Search using keywords included in the card.<br/>* Only CSN cards and Wiegand cards can be searched. |
    | <Cmd sid="setting.server.custom_field.title" /> | Search using keywords included in the <Cmd sid="setting.server.custom_field.title" />.<br/>* <Cmd sid="setting.server.custom_field.title" /> searches are case-sensitive.<br/>* If <Cmd sid="setting.sever.secure.comm.encrypt.db" /> is enabled, it can only be searched in full text. |
    | <Cmd sid="button.advancedSearch.clearAll" /> | Press the button to clear or reset all the fields in the form to their default or empty values. |

    </div>

    :::info

    When searching using multiple fields, only users that match all search words are searched.

    :::

1. Enter the desired keywords in the desired field and click <Cmd sid="button.advancedSearch.search" />. Users matching search words will be searched.
