## New Features and Improvements

---

1. Discontinuation of support for the BioStar 2 **VIDEO**.

1. Supports **Custom User Field** - **File Upload** option that allows files to be uploaded to user information.

1. Added **Device Display Message** UI to enter a private message to be displayed on the device screen during user authentication.

1. Added an option to set face authentication to deny authentication if a mask is worn.

1. Added a separate management page for **Deleted CSN Mobile Card**.

1. Updated **Timed Anti PassBack** feature.

    * Improved UI to allow setting **Timed Anti Passback** for each device.

    * Added a **Bypass Group** for **Timed Anti Passback**.

1. Added **Firmware Version** to device list.

1. Improved **DOOR** menu UI.

    * Improved door **Open Time** setting UI.

    * Improved UI in the alarm settings to differentiate between **Anti PassBack** and **Timed Anti PassBack**.

1. Improved **REPORT** performance.

    * Merged report service into AC service.

    * Changed filter condition from AND to OR when creating **Custom Report**.

1. Improved **DASHBOARD** performance.

1. Added new predefined Wiegand card formats.

    * MIFARE CSN 32bit

    * MIFARE CSN 34bit (Parity)

    * DESFire 56bit

    * DESFire 58bit (Parity)

1. Improved **Advanced Search**.

    * Improved user list printing feature.

    * Improved to allow selecting all after **Advanced Search**.

1. Added **Elevator** and **Zone** columns to the **Column List** of the **Event Log**.

1. Upgraded Bootstrap library (3.4.1).

1. Improved USB Agent security vulnerabilities.

1. Improved performance of **Schedule** in **TIME ATTENDANCE** menu.

1. Improved performance of **Report** in **TIME ATTENDANCE** menu.

1. Upgraded Node.js version.

1. Improved to ensure that calling the TA API retrieves correct result values according to the set user permissions.

1. Improved security vulnerabilities.

    * Improved HTTPS protocol security vulnerabilities.

    * Improved SQL Injection security vulnerabilities.

1. Improved the recognition rate of BioStar 2 QR.

1. Optimized performance when updating users on the C server.

1. Modified the message displayed if validation fails when creating a CSN card using API.

1. Improved performance when enrolling cards.
