Release<Separator/>2023-09-20

## New Features and Improvements

---

1. Supports a new device.

    * BioStation 2a

1. Added **Template on Mobile** feature.

    * BioStation 3 FW 1.2.0 or higher

1. Supports U&Z Wireless Door Lock connection to CoreStation.

1. Supports **Custom Smart Card Layout**.

    * Xpass D2 FW 1.7.1 or higher

1. Improved **Smart Card Layout** settings.

1. Improved **Administrator** menu UI on the **DEVICE** detail page.

1. Improved UX of **Use for BioStar 2 Login** menu in **ACTIVE DIRECTORY**.

1. Updated OpenSSL.

1. Updated OpenJDK.

1. Improved SQL Injection vulnerability.

1. Improved security vulnerabilities in **Active Directory** account settings.

1. Separated ID storage for **Active Directory** login and BioStar 2 login.

1. Improved security vulnerabilities in video packages.

1. Improved **Audit Trail** to be logged when a user attempts to log in with an incorrect password and exceeds the set Maximum Invalid Attempts.

1. Blocked modifiyng **Login ID** to '**1**', the default ID of the initial administrator.

    * After upgrading to BioStar 2 v2.9.4, cannot log in using the administrator account with **Login ID** set to '**1**'.

1. Improved **User Group** sync performance.

1. Improved performance when **Automatic User Synchronization** is set to **Specific Devices(Only devices belonging to the access group)** option.

    * Improved **Access Group** edit performance.

    * Improved device **Batch Edit** performance.

    * Improved **Access Level** and **Floor Level** creation performance.

    * Improved **DOOR** edit and deletion performance.

1. Improved sync performance when device gets reconnected to server.

1. Improved user search performance in **VISITOR**.

1. Blocked issuing **Secure Credential** when **Card ID** starts with 0.

1. Improved device initialization guidance message not to be displayed if the Smart Card Layout set on the device does not exist in BioStar 2.

1. Improved to stop requesting logs to a device if it sends abnormal event logs to the server more than 5 times.

1. Added to check for duplicate user IDs or login IDs while decrypting data by setting **Encrypt Personal Data on Database** to **Not Use**.

1. Deleted unsupported ports from the **Port** menu.

    * **Report Cloud Port**

1. Improved to display of supported special characters in a pop-up message when a user name contains unsupported characters.

1. Improved the error message displayed when connecting a dormant account in the Airfob Portal or an account using private authentication mode to Mobile Access in BioStar 2.

1. Improved error message appears when attempting to delete a disconnected device from the **MOBILE ACCESS**.

1. Changed device names

    * Xpass 2 → XPass 2

    * Xpass D2 → XPass D2

1. Improved Japanese in **Send Visual Face Mobile Enrollment Link** mail.
