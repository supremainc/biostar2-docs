Release<Separator/>2022-09-26

## New Features and Improvements

---

1. Supports BioStation 3

1. Supports user information partial update

    * Supporting device: BioStation 3 v1.0.0

1. Improved extraction of images used for visual face credentials

1. Simplified visual mobile face enrollment process

1. Supports visual face import

    * Enrolls visual face when file name of the image matches user ID

1. Supports visual face migration

1. Device added supporting Fingerprint LFD

    * Supporting device: BioLite N2 v1.5.0

1. Improved door open logs

1. Added logs to indicate full capacity on devices

1. Improved security vulnerabilities

    * Improved Unquoted Service Path vulnerability

    * Improved Clickjacking vulnerability

    * Improved minor vulnerabilities

1. CSV Improved error messages on CSV import fail
