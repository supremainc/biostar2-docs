Release<Separator/>2023-02-08

## New Features and Improvements

---

1. Supports 3rd party OSDP reader connection.

1. Supports activating licenses for use of specific features on devices.

1. Improved **Face Detection Setting**.

    * **Detection Distance**, **Wide Search** option support.

1. Improved security vulnerabilities

    * Updated jQuery to the latest version of help menu

    * Improved security vulnerabilities for administrator permissions

    * Improved HTTP vulnerability

    * Improved Injection vulnerability

    * Improved Authentication vulnerability

    * Improved minor vulnerabilities

1. Improved saved file name of CSV Export.

1. Improvement to use only the directory that has already been created when setting the storage path for the image log.

1. Improved SMTP setting security vulnerabilities.

1. Added case insensitivity guide for databases and tables in custom installation.

1. Improved Time & Attendance security vulnerabilities

    * Improved HTTP Content-Security-Policy, X-XSS-Protection, X-Content-type-options vulnerability

    * Improved TA Path Traversal vulnerability

1. Improved real-time event log performance.

1. Improved performance on adding users and user groups.

1. Improved performance on loading unacknowledged alarms.

1. Improved peformance for high-volume environments with many users.

1. Improved encryption/decryption data processing performance.

1. Improved BioStar 2 logout speed.

1. Optimized memory settings for Maria DB when installing BioStar 2.

1. Improved pop-up message when checking image logs of the disconnected device.

1. Improved pop-up message when setting an elevator unlock schedule with a 1st generation device.

1. Improvemed photo deterioration due to noise when uploading photos for enrolling Visual Face.

1. Improved to check image data to be encoded when enrolling Visual Face using API.

1. Improved the SMTP information of **Trigger & Action** and **Email Setting** to be synchronized with each other.

1. Improved visitor search performance in an environment using MS SQL Server database.

1. Added new login page.

1. Minor string fixes.

1. Minor performance improvements.

1. Minor stabilization of server.
