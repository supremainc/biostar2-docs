Release<Separator/>2023-06-26

## New Features and Improvements

---

1. Added **REPORT** feature.

1. Added **CARD PRINTER** feature.

    * Supports card printing directly from BioStar 2 using the cardPresso.

1. Added **SYSTEM BACKUP** and **RESTORE** feature.

    * Provides **BioStar Restore** program that can restore the system using backup files (included in BioStar 2 installer).

1. Supports QR code authentication in BioStation 3.

    * Firmware v1.1.0 or higher.

1. Supports Xpass D2 **Keypad Backlight**.

    * XPass D2 (XPD2-GKDB) hardware V02M, firmware v1.7.1 or higher.

1. Improved security vulnerabilities.

    * Update Moment.js library version.

    * Updated Lodash library version.

    * Improved enckey file creation logic.

1. Improved update speed on **CSV Import** when using **Encrypt Personal Data on Database**.

1. Improved to properly recognize characters using UTF-16 in CSV files when **CSV Import**.

1. Improved Time & Attendance security vulnerabilities.

    * Swagger UI Version upgrade.

    * Added X-Frame-Options.

    * Improvement of iframe vulnerability by providing Unified Gateway feature.

1. Improved pop-up messages that appear when adding users to the **TIME ATTENDANCE** menu when exceeding the maximum number of users set by license type.

1. Improved performance when **Automatic User Synchronization** is set to **Specific Devices(Only devices belonging to the access group)** option.

    * Improved device setup performance.

    * Improved processing performance on zone activation/inactivation.

    * Improved synchronization performance when moving user groups.

1. Improved inefficient DB check logics when executing queries on the server.

1. Improved performance by adding missing indexes to temporary tables.

1. Optimized subquery when updating or deleting temporary tables using the User API.

1. Improved **Door Status** inquiry performance of **MONITORING**.

1. Improved to check whether unsupported characters are used in user names when enrolling mobile access card.

1. Supports entering symbols (+, -, @, .) in configuration of the **Intercom** menu.

    * **SIP Username**, **Authorization ID**, **Extension Number**

1. Added event code related to QR code authentication.

1. Added a guide message to the existing license menu location (**Settings** â†’ **SERVER**) to prevent location confusion due to the new **LICENSE** menu.

1. Improvement of error message that appears when enrolling mobile access card when an **Email** is not entered in the user information.

1. Improved visibility of icons for each device in the device list.

1. Updated login page.
