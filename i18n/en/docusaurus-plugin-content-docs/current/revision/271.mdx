Release<Separator/>2018-12-20

## New Features and Improvements

---

1. Supports synchronization of all devices connected to the server when users update the device.

1. Improves the ability to delete users on a device from the server's user menu.

1. Support to Input(Event Name Change) of Trigger & Action in the device advanced setting.

1. Changes the Java version.

    * Before: 1.8.0_161
    
    * After: 1.8.0_191

1. Provides the log about whether a user has been updated on the device or the server.

1. Adds root password verification procedures during installation and upgrade.

## Main Fixes

---

1. Fixed a security vulnerability: Cross Site Scripting.

1. The row setting changes when the page is refreshed after setting the number of rows in the monitoring menu.

1. The log does not display normally when a user moves the page back and forth after setting the event log filter in the Monitoring menu.

## Bug Fixes

---

1. Support to the OM-120 Relay time setting value from 1 second.
