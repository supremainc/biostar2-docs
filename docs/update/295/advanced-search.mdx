기존 <Cmd sid="common.gnb.user" /> 메뉴의 검색 기능은 이름과 이메일로만 사용자를 검색할 수 있었으나 새롭게 추가된 <Cmd sid="button.advancedSearch" /> 기능에서는 검색어의 범위를 줄이고 관련 없는 정보를 제외하여 원하는 정보를 정확하게 검색할 수 있습니다.

## 고급 검색 사용하기

1. <Cmd sid="common.gnb.user" /> 메뉴를 클릭하세요.

1. <Cmd sid="button.advancedSearch" />을 클릭하세요.

    <Image src='/img/ko/biostar-user-menu.png' caption />

    <Image src='/img/ko/biostar-295-user-adv-search.png' className='none' caption />

    <div className='overflow-x'>

    | 항목 | 설명 |
    | -- | -- |
    | <Cmd sid="common.id" /> | 사용자의 ID에 포함된 키워드로 검색할 수 있습니다. |
    | <Cmd sid="common.name" /> | 사용자의 이름에 포함된 키워드로 검색할 수 있습니다.<br/>* <Cmd sid="common.name" /> 검색은 대소문자를 구분하지 않습니다. |
    | <Cmd sid="common.email" /> | 사용자의 이메일에 포함된 키워드로 검색할 수 있습니다. |
    | <Cmd sid="common.department" /> | 사용자가 소속된 부서명이 포함된 키워드로 검색할 수 있습니다.<br/>* <Cmd sid="setting.sever.secure.comm.encrypt.db" />가 활성화되어 있는 경우에는 전문 검색만 가능합니다. |
    | <Cmd sid="common.title" /> | 사용자의 직함이 포함된 키워드로 검색할 수 있습니다.<br/>* <Cmd sid="setting.sever.secure.comm.encrypt.db" />가 활성화되어 있는 경우에는 전문 검색만 가능합니다. |
    | <Cmd sid="common.telephone" /> | 사용자의 전화번호가 포함된 키워드로 검색할 수 있습니다. |
    | <Cmd sid="common.group" /> | 선택한 그룹에 속한 사용자를 검색할 수 있습니다. |
    | <Cmd sid="common.accessGroup" /> | 선택한 출입 그룹에 속한 사용자를 검색할 수 있습니다. |
    | <Cmd sid="common.status" /> | <Cmd sid="user.status.active" />/<Cmd sid="user.status.inactive" /> 상태에 해당하는 사용자를 검색할 수 있습니다. |
    | <Cmd sid="user.bioStarOperator" /> | 선택한 권한 등급에 해당하는 사용자를 검색할 수 있습니다. |
    | <Cmd sid="common.card" /> | 카드에 포함된 키워드로 검색할 수 있습니다.<br/>* CSN 카드, Wiegand 카드만 해당합니다. |
    | <Cmd sid="setting.server.custom_field.title" /> | <Cmd sid="setting.server.custom_field.title" />에 포함된 키워드로 검색할 수 있습니다.<br/>* <Cmd sid="setting.server.custom_field.title" /> 검색은 대소문자를 구분합니다.<br/>* <Cmd sid="setting.sever.secure.comm.encrypt.db" />가 활성화되어 있는 경우에는 전문 검색만 가능합니다. |
    | <Cmd sid="button.advancedSearch.clearAll" /> | 모든 필드에 입력된 값을 삭제하거나 기본값 또는 빈칸으로 재설정할 수 있습니다. |

    </div>

    :::info

    복수의 필드를 이용하여 검색한 경우에는 검색 조건이 모두 일치하는 사용자만 검색됩니다.

    :::

1. 검색을 원하는 필드에 검색할 키워드를 입력하고 <Cmd sid="button.advancedSearch.search" />을 클릭하세요. 검색 조건이 일치하는 사용자가 검색됩니다.
