커스텀 사용자 필드를 사용하여 사용자 정보에 파일을 업로드하고 저장할 수 있습니다.

1. <Cmd sid="common.setting" /> > <Cmd sid="common.server" /> > <Cmd sid="setting.server.userDeviceManagement" />로 이동하세요.

1. <Cmd sid="setting.server.custom_field.title" />의 <Image src='/img/ko/ico-add.png' ico />를 클릭하여 필드를 추가한 다음, 원하는 이름을 입력하고 <Cmd sid="setting.server.custom_field.configure.th.type" />을 <Cmd sid="setting.server.custom_field.type.fileupload" />를 선택한 후 <Cmd sid="button.apply" />을 클릭하여 저장하세요.

    <Image src='/img/ko/biostar-custom-user-field-file-upload.png' className='none' caption />

    :::info

    파일 업로드 필드는 최대 2개까지 추가할 수 있습니다.

    :::

1. <Cmd sid="common.gnb.user" /> 메뉴로 진입한 다음 사용자 목록에서 원하는 사용자를 클릭하세요.

1. 추가한 <Cmd sid="setting.server.custom_field.type.fileupload" /> 커스텀 사용자 필드 부분에 <Image src='/img/ko/ico-browse.png' ico />를 클릭하여 원하는 파일을 업로드하세요.

    <Image src='/img/ko/biostar-custom-user-field-file-upload-1.png' caption />

    :::note

    **업로드 파일 제약사항**

    * 파일 유형에는 제약이 없습니다.

    * 지원하는 최대 파일 크기는 1MB 입니다.

    * 파일명은 영숫자와 특수 기호(-, _)을 포함하여 최대 120자까지 입력할 수 있습니다.

    :::

1. 업로드한 파일을 저장하려면 <Cmd sid="button.apply" />을 클릭하고, 삭제하려면 <Image src='/img/common/ico-upload-delete.png' ico />를 클릭하세요.

    * 업로드한 파일을 저장한 후에는 사용자 정보에서 <Image src='/img/common/ico-download.png' ico />를 클릭하여 파일을 다운로드하거나 <Image src='/img/common/ico-delete.png' ico />를 클릭하여 삭제할 수 있습니다.

    <Image src='/img/common/biostar-298-custom-user-field-file-upload-2.png' caption />
