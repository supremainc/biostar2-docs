## 신규 기능 및 개선

---

1. 신규 장치 연결 지원

    * XPass 2 (XP2-MAPB-H)

1. 신규 컨트롤러 연결 지원

    * CoreStation 20 (CS-20, CS-20P), DoorInterface (DI-24)

1. <Cmd sid="device.info.keypadBackLight" /> 설정 기능 지원

    * XPass 2 (XP2-GKDPB) 하드웨어 v1.2.0 이상, 펌웨어 v1.4.3 이상

1. Microsoft Entra ID(Azure Active Directory) 연동하여 BioStar 2에 로그인할 수 있도록 지원

1. <Cmd sid="settings.activeDirectory" /> 동기화 성능 개선

1. 인증 메뉴에 스크램블 키패드를 켜고 끌 수 있는 옵션 추가

1. 보안 취약점 개선

    * 파일 업로드 및 입력값에 대한 검증 강화

1. MS SQL Server 2022 및 OLE DB 드라이버 지원

1. MS SQL 보안 통신 TLS 1.2 지원

1. MariaDB 버전 업그레이드 (11.4.4)

    * Windows Server 2012 운영 체제에서는 MariaDB 11.4.4 버전이 지원되지 않으므로, BioStar 2를 2.9.10 버전으로 업그레이드할 수 없습니다.

1. 근태 관리 보안 취약점 개선

    * External service interaction (HTTP) 보안 취약점 개선

1. 시스템 로그인 로직 개선

1. 근태 관리 /punch_logs/modified API 사용성 개선

1. MSSQL 윈도우 인증 사용 시 DB 연결이 끊긴 후 서버 재시작 없이도 자동으로 재연결되도록 개선

1. Microsoft ODBC Driver 17 for SQL Server 지원

1. 사용자 동기화 성능 개선

1. AC 서비스를 가동하고있지 않은 경우에는 동기화를 진행하지 않도록 개선

1. 컨트롤러가 <Cmd sid="device.card.customSmartCard" />를 지원하지 않더라도, 연결된 슬레이브 장치가 해당 기능을 지원하는 경우 설정이 가능하도록 개선

1. 암복호화 마이그레이션 실행 시 장시간 경과로 TA DB 연결이 끊긴 경우, 자동으로 재연결되도록 개선

1. 출입 그룹에 속해 있는 사용자 조회 성능 개선

1. TCP/IP로 연결된 슬레이브 장치에서 사용자에게 크리덴셜을 추가하려고 할 때 나타나는 팝업 메시지 개선

1. 오프라인 환경에서 라이선스 활성화에 실패할 경우 에러 코드를 포함한 팝업 메시지가 표시되도록 개선

1. <Cmd sid="device.info.interphone" /> 설정의 <Cmd sid="device.intercom.account.new" />과 <Cmd sid="device.intercom.authorization.id" />에 일부 특수문자(`*`, `,`, `_`) 지원

1. <Cmd sid="audit.menu.video" /> 메뉴 성능 개선

1. 일부 슬레이브 장치에서 <Cmd sid="device.info.tamperOn" /> 설정이 표시되도록 개선

1. 클라우드 등록 시 네트워크 환경에 따라 발생할 수 있는 오류 메시지를 안내하여 원인 확인이 가능하도록 개선

1. <Cmd sid="audit.menu.setting.cloud" /> 계정을 <Cmd sid="common.notUse" />으로 전환할 수 있는 기능을 추가하고, 재등록 메일 발송, 상태 정보 표시, 연결 포트 검사 기능을 지원하도록 개선

1. <Cmd sid="audit.menu.systemBackup" /> 진행 시 <Cmd sid="monitoring.auditTrail" />에 성공 또는 취소 상태가 표시되도록 개선

1. BioMini 제품 종류에 따라 아이콘이 표시되도록 개선
