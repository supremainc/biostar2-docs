## 주요 버그 수정

---

<BugLists>

1. 다수의 FaceStation F2가 동시에 연결될 때 간헐적으로 확장 인증 모드 설정이 삭제되는 문제 <Badge>발생 버전: v2.8.8</Badge>

1. **모든 이력** 옵션을 선택해 업데이트한 개인별 보고서에 시간 기록 종류 값이 Null인 시간 기록이 표시되지 않는 문제 <Badge>발생 버전: v2.7.10</Badge>

1. **모든 이력** 옵션을 선택해 업데이트한 개인별 보고서에 출입 기록이 정상적으로 기록되지 않는 문제 <Badge>발생 버전: v2.7.10</Badge>

    * 사용자가 근태관리 장치에서 **퇴근**을 선택해도 근태 보고서에는 **출근** 로그만 기록

1. MS SQL Windows 인증으로 BioStar 2를 설치하였을 때 기본 포트 이외의 포트를 사용할 경우 TA 데이터베이스에 연결할 수 없는 문제 <Badge>발생 버전: v2.7.14</Badge>

1. 장치 기본 해시 키 전송 오류로 인해 PIN 인증에 실패하는 문제 <Badge>발생 버전: v2.4.0</Badge>

1. 얼굴을 등록한 사용자와 비주얼 페이스를 등록한 사용자를 FaceStation 2와 FaceStation F2에 동시에 전송할 경우 FaceStation F2에서는 얼굴 인증에 실패하는 문제
<Badge>발생 버전: v2.8.6</Badge>

1. 근태관리 장치 동기화 문제

    * 장치 메뉴에 등록된 장치가 근태관리 메뉴에 동기화되지 않는 문제
    
    * 장치 메뉴에 등록된 FaceLite가 근태관리 장치 목록에 표시되지 않는 문제 <Badge>발생 버전: v2.7.10</Badge>

    * 동기화 문제로 인해 장치 그룹 ID가 Null 값으로 처리되어 장치 메뉴에 등록된 장치가 근태관리 장치 목록에 표시되지 않는 문제 <Badge>발생 버전: v2.7.5</Badge>

1. T&A 서비스를 시작하면 BioStar 2에서 자동 로그아웃되고 로그인이 불가능한 문제 <Badge>발생 버전: v2.8.4</Badge>

    * BioStar 2 로그인되었을 때 T&A 서비스에 접근할 경우 **Unknown error. Check the log file for detailed information** 오류 메시지 출력

1. 모바일 출입카드 **사용** 설정 후 서버 메뉴에서 확인을 클릭하면 모바일 출입카드 설정이 **사용 안 함**으로 변경되는 문제 

1. 이메일 자동 발신을 위한 SMTP를 정상적으로 설정하였음에도 경고 근무 시간 보고서가 발송되지 않는 문제 <Badge>발생 버전: v2.8.6, v2.8.8</Badge>

1. 개인 정보 DB 암호화를 켜거나 끈 상태에서 사용자 동기화를 실행한 뒤 근무 기록을 생성하면 오류가 발생하는 문제 <Badge>발생 버전: v2.8.0</Badge>

1. MS SQL Server 데이터베이스를 사용하는 환경에서 다수의 사용자 그룹에 대한 권한을 가진 커스텀 권한 관리자로 로그인할 경우 BioStar 2가 메모리를 과도하게 점유하는 문제 <Badge>발생 버전: v2.8.7</Badge>

1. 언어를 영어로 설정했을 때 모바일 카드 발급 시 표시되는 팝업 메시지에 장치 모델명이 잘못 표시되는 문제 <Badge>발생 버전: v2.6.0</Badge>

1. FaceStation F2의 온도 관련 이벤트 로그를 업로드할 때 시간이 오래 걸리고, 온도 값이 0인 데이터와 같이 불필요한 로그까지 업로드하는 문제 <Badge>발생 버전: v2.8.6</Badge>

</BugLists>
