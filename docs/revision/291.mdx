## 신규 기능 및 개선

1. 타사 OSDP 리더 연결 지원

2. 장치에서 특정 기능 사용에 대한 라이선스 활성화 지원

3. **얼굴 검출 설정** 기능 개선

    * **얼굴 인식 거리**, **얼굴 인식 영역 넓히기** 옵션 지원

4. 보안 취약점 개선

    * 도움말 jQuery 최신 버전 적용

    * 관리자 권한에 대한 보안 취약점 개선

    * HTTP 취약점 개선

    * Injection 취약점 개선

    * 인증 취약점 개선

    * 기타 보안 취약점 개선

5. 특정 CSV 내보내기 실행 시 저장되는 파일명 개선

6. 이미지 로그의 저장 경로 설정 시 이미 생성되어있는 디렉토리만 사용할 수 있도록 개선

7. SMTP 설정 보안 취약점 개선

8. 사용자 정의 설치 시 데이터베이스 및 테이블의 대소문자 미구분 가이드 추가

9. 근태 관리 보안 취약점 개선

    * HTTP Content-Security-Policy, X-XSS-Protection, X-Content-type-options 취약점 개선

    * TA Path Traversal 취약점 개선

10. 실시간 이벤트 로그 성능 개선

11. 사용자 및 사용자 그룹의 생성 속도 개선

12. 미확인 알람 조회 시 로딩 속도 개선

13. 사용자가 많은 대용량 환경에서 일부 기능 속도 개선

14. 암복호화 데이터 처리 성능 개선

15. BioStar 2 로그아웃 속도 개선

16. BioStar 2 설치 시 Maria DB 메모리 설정 최적화

17. 연결되어있지 않은 장치의 이미지 로그 확인 시 나타나는 팝업 메시지 개선

18. 1세대 장치로 엘리베이터 스케줄 개방 설정 시 나타나는 팝업 메시지 개선

19. 사진을 업로드하여 비주얼 페이스 등록 시 노이즈로 인한 사진 화질 저하 개선

20. API를 이용하여 비주얼 페이스 등록 시 인코딩 여부 확인하도록 개선

21. **작업 조건 및 동작**과 **이메일 설정**의 SMTP 정보가 서로 동기화 되도록 개선

22. MS SQL Server 데이터베이스를 사용하는 환경에서 방문자 검색 성능 개선

23. 로그인 페이지 변경

24. 일부 용어 변경

25. 기타 성능 개선

26. 기타 서버 안정화

## 버그 수정

1. 특정 사용자 그룹에서 사용자를 전체 체크하고 **사용자 삭제** 시 다른 그룹에 포함된 사용자의 모바일 카드까지 삭제되는 문제 (발생 버전: v2.7.12)

2. 특정 라이브러리가 Windows Server 2012에서 설치되지 않는 문제 (발생 버전: v2.9.0)

3. API를 이용하여 1000명 이상의 사용자를 삭제할 경우 사용자가 삭제되지 않는 문제 (발생 버전: v2.8.0)

4. API를 이용하여 사용자를 추가하고 start_datetime 및 expiry_datetime에서 시간을 설정하는 경우 정확한 시간이 반영되지 않는 문제 (발생 버전: v2.0.0)

5. 등록된 사운드 파일 삭제 시 필요한 권한이 있어도 거부되어 삭제할 수 없는 문제 (발생 버전: v2.8.14)

6. 특정 장치의 아이콘이 제대로 표시되지않는 문제 (발생 버전: v2.3.0)

7. 간헐적으로 장치 연결이 끊어지는 문제 (발생 버전: v2.0.0)

8. 등록된 사용자가 많은 상태에서 사용자 그룹 정보를 변경하면 변경점이 반영되지 않는 문제 (발생 버전: v2.7.14)

9. 동일한 출입레벨을 사용하는 출입그룹 중 사용자 그룹이 설정된 출입그룹 삭제 시 사용자 그룹에 포함된 사용자가 단독으로 추가되어있는 다른 출입그룹에서도 삭제되는 문제 (발생 버전: v2.8.14)

10. 서버 재기동 이후에 발생되는 **장치연결해제 감지** 이벤트가 일부 누락되는 문제 (발생 버전: v2.8.11)

11. FaceStation F2의 장치 로그가 간헐적으로 누락되는 문제 (발생 버전: v2.8.6)

12. 장치 연결 시 간헐적으로 로그인 페이지로 이동되는 문제 (발생 버전: v2.9.0)

13. MS SQL Server 데이터베이스를 사용하는 환경에서 사용자 정보의 이메일 주소를 이미 사용중인 이메일 주소로 변경하여도 중복 체크되지 않는 문제 (발생 버전: v2.8.16)

14. 비주얼 페이스 모바일 등록 안내 이메일 발송 시 스페인어 바닥글이 표시되지 않는 문제 (발생 버전: v2.8.16)

15. **CSV 불러오기**를 이용하여 CSN 모바일 카드 발급 시 유효 기간에 슬래시(/)를 사용한 경우 올바르게 표기되지 않는 문제 (발생 버전: v2.8.9)

16. BioStar 2 로그인 시 일부 구 버전 장치에 대한 펌웨어 업그레이드 고지가 나타나지 않는 문제 (발생 버전: v2.8.6)

17. **사용자 자동 동기화를 특정 장치(해당하는 출입 그룹에 속한 장치만 동기화)**로 설정 시 **CSV 불러오기**로 업데이트한 사용자 정보가 장치에 동기화 되지 않는 문제 (발생 버전: v2.7.11)

18. API를 이용하여 카드 발급 시 최대 발급 개수를 초과하여 발급이 가능한 문제 (발생 버전: v2.0.0)

19. **액티브 디렉토리**에서 **BioStar 2 로그인에 사용** 옵션이 비활성화 상태일 때 액티브 디렉토리 동기화 시 로그인되어있는 ID가 동기화되는 문제 (발생 버전: v2.8.16)

20. 다수의 비주얼 페이스가 등록된 사용자를 데이터 내보내기하면 서버가 멈추는 문제 (발생 버전: v2.8.10)

21. CSV 파일의 사용자 정보에 최대 입력 글자 수가 초과하지 않아도 **CSV 불러오기**에 실패하는 문제 (발생 버전: v2.8.16)

22. MS SQL Server 데이터베이스를 사용하는 환경에서 특정 언어로 부재가 등록된 경우 근태 기록 조회 시 해당 언어가 올바르게 표시되지 않는 문제 (발생 버전: v2.2.1)

23. **사용자** 메뉴에서 다수의 사용자 그룹을 여러 번 이동한 경우 제대로 동기화가 되지 않는 문제 (발생 버전: v2.8.9)

24. 다수의 **시간 규칙**이 생성되어있는 경우 **근태 관리**에 접근이 되지 않는 문제 (발생 버전: v2.2.1)

25. MS SQL Server 데이터베이스를 사용하는 환경에서 긴 내용의 인증서를 적용한 경우 **근태 관리**에 접근이 되지 않는 문제 (발생 버전: v2.2.1)

26. **근무 스케줄**에 새 사용자를 추가하는 과정에서 스케줄에 등록된 사용자가 초기화 되는 문제 (발생 버전: v2.4.0)
