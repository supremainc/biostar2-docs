### 릴리즈

2020-10-29

## 신규 기능 및 개선

1. CSV 불러오기를 이용한 비주얼 페이스 일괄 등록 지원

1. 열화상 카메라 기능 안정화

    * 알림 메일에 온도 표기 추가
    
    * 모니터링에 열화상 보고서 추가

1. **특정 장치** 사용자 자동 동기화 옵션 사용 시 실패한 전송에 대한 재시도 횟수를 3회로 제한
  
1. 1차 BioStar 2 성능 개선

    * CSV 불러오기 속도 향상

    * 데이터베이스 부하로 인한 속도 저하 개선

    * 사용자, 장치, 출입문, 모니터링 메뉴의 화면 로드 및 조회 속도 향상

    * 카드 조회 속도 향상

    * 대시보드 진입 시 로그 조회 빈도를 주 단위로 변경하여 로그인 속도 개선

1. New API 2.8.7 기준 보완

    * User API 쿼리를 호출할 때 기간 옵션이 between인 경우 parameter 크기 확인 로직이 누락되는 문제 개선

    * Swagger API 기능 안정화

    * GET/api/cards를 이용해 사용자 카드 조회 시 Status 값이 비정상적으로 출력되는 문제 개선

1. BioStar 2 설치 시 db-converter를 백그라운드에서 실행하도록 개선
  
1. 로그인 시 암호화 키 일치 여부를 확인하여 불일치할 경우 오류 메시지를 출력하도록 개선

1. BioStar 2 설치 시 데이터베이스 연결 확인 로직 개선
  
1. 액티브 디렉토리에 사용자 동기화 예외 처리 개선

1. 로그에 로그인 권한 정보가 표시되지 않도록 개선
  
1. 이벤트 로그 및 실시간 로그 목록의 온도 열을 기본값에서 제외

1. 비주얼 페이스 모바일 등록 메일 내용 개선

## 주요 버그 수정

1. 이벤트 로그에서 슬레이브 장치는 이벤트 필터를 적용하여 조회할 수 없는 문제

1. 개인 정보 DB 암호화를 사용할 때 CSV로 모바일 출입카드를 발급할 경우 미할당된 CSN 카드와 중복된 카드 ID가 존재하면 오류가 발생하는 문제

1. 모바일 출입카드 사용을 설정하지 않았음에도 CSV로 모바일 출입카드가 등록되는 문제

1. 서버를 다시 시작하면 사용자 자동 동기화 설정이 기본값으로 변경되는 문제

1. 개인 정보 DB 암호화를 사용할 때 이미지 로그 저장 경로를 변경할 경우 변경된 경로의 이미지 로그 파일이 암호화되지 않는 문제

1. BioStar 1.x to BioStar 2.x Migration Tool을 이용해 마이그레이션을 실행하였을 때 일부 사용자의 카드가 누락되는 문제

1. **특정 장치** 사용자 자동 동기화 옵션 사용 시 동기화 오류로 인해 사용자가 삭제되는 문제

1. **특정 장치** 사용자 자동 동기화 옵션 사용 시 비주얼 페이스를 등록한 사용자가 동기화되지 않는 문제

1. 장치에 언어 리소스가 정상적으로 적용되지 않는 문제 (FSF2-ODB)

## 버그 수정

1. CoreStation 장치 상세 페이지에서 카드 사용을 설정한 뒤 재진입하면 화면상에 카드 설정이 **사용 안 함**으로 출력되는 문제

1. Linux 환경에서 MariaDB를 사용할 때 테이블명의 대·소문자를 구분하여 쿼리문이 실행되도록 수정

1. 클라우드 접속 후 모니터링 메뉴에 진입할 때 잘못된 주소를 참조하여 시간이 오래 걸리는 문제

1. System.conf 파일이 깨질 때 enckey 파일도 깨지는 문제

1. 사용자 ID 종류를 영숫자로 설정하고, 스마트 카드의 종류를 액세스 온 카드로 설정했을 때 카드 ID를 자동으로 불러오지 못하는 문제

1. 사용자에 다수의 출입 그룹이 적용되었을 때 사용자 목록에서 해당 사용자의 출입 그룹 표기 순서가 매번 바뀌는 문제

1. Video service 구동 관련 오류

    * BioStar 2 설치 시 Video service 구동 여부에 따라 redis service를 구동하도록 수정

    * Video service 구동 시 SSL 사용 여부에 따라 http 또는 https service를 구동하도록 수정

1. 확장 인증 모드가 데이터베이스를 포함한 BioStar 2에서 삭제되는 문제

1. 클라우드를 비활성화한 상태에서 비주얼 페이스 모바일 등록 링크 전송을 실행할 때 발생하는 오류 메시지의 텍스트가 깨지는 문제

1. 비주얼 페이스 이미지 비율 수정

1. 방문자 지문 스캔을 시도할 때의 지연 현상 개선

1. 일부 근태관리 기능 오류 수정

    * 시간 기록 로그 수정이 불가능한 문제

    * 개인별 보고서 출력 시 로딩이 무한하게 지속되는 문제

1. 모바일 출입카드 사용 장치를 20대 이상 등록해도 장치 목록에 20대까지만 보여지는 문제
