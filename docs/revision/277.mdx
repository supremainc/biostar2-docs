## 신규 기능 및 개선

---

1. 방문자 출입 관리 기능 추가

1. 근태 관리 라이선스 정책 변경

1. 사설 도메인 주소를 이용해 BioStar 2에 접속해도 근태 관리 메뉴를 사용할 수 있도록 개선

1. 사용자가 근태 관리 메뉴에 동기화될 때 로그인 ID를 가진 사용자부터 동기화가 진행되도록 개선

1. BioStar 2 설치 과정에서 dbscript 오류가 발생할 경우 BioStar 2 설치 완료 화면에 **Error dbscript db-converter** 메시지 제공

1. Java 버전 변경

    * 기존: 1.8.0_201 버전

    * 변경: 1.8.0_212 버전

## 주요 버그 수정

---

1. 스마트 카드 레이아웃 사용 중 레이아웃 옵션을 **없음**으로 변경해도 설정이 적용되지 않는 문제

1. MS SQL Server를 재시작하면 데이터베이스가 정상적으로 연결되지 않는 문제

1. 이미지 로그의 이벤트 목록에서 **소프트 안티패스백 위반 감지**, **하드 안티패스백 위반 감지** 항목이 누락된 문제

1. 일부 장치(BS2-OIPW, BSA2-OIPW)에서 RFID 카드를 지원 옵션과 다르게 지원하는 문제

1. 커스텀 사용자 필드를 구성한 경우, 사용자 목록을 인쇄할 때 필드 이름은 출력되나 데이터는 표시되지 않는 문제

1. 장치 설정에서 관리자를 설정한 경우, 장치를 초기화해도 관리자 설정은 초기화되지 않는 문제

1. 액티브 디렉토리 암호화를 설정할 때 루트 인증서를 복사한 뒤 서버를 재시작하지 않으면 Active Directory 서버와 연결되지 않는 문제

1. BioStar 2.3.0 이하 버전에서 생성한 커스텀 권한 관리자가 있을 경우, 2.6.3 이상 버전에서 커스텀 권한 목록이 출력되지 않는 문제

1. 스마트 카드 레이아웃의 설정을 변경할 때 변경 사항이 있는 카드 유형의 레이아웃만 동기화가 진행되는 문제

1. MS SQL Server 데이터베이스를 사용하는 환경에서 이벤트 로그를 수동 또는 자동으로 가져올 때 로그 중 사용자 ID가 32자인 데이터가 있을 경우 오류가 발생하는 문제

1. 이벤트 로그를 수동 또는 자동으로 가져올 때 일부 로그가 누락되는 문제

## 버그 수정

---

1. 사용자 인터페이스가 정상적으로 출력되지 않는 문제

    * 얼굴 그룹 매칭 설정에서 사용자 그룹을 선택할 때 얼굴 아이콘 및 문구가 비정상적으로 출력되는 문제

    * 계정 설정의 아이템 상세 권한 화면이 비정상적으로 출력되는 문제

    * 계정 설정의 아이템 상세 권한에서 아이템을 선택할 때 문구가 비정상적으로 출력되는 문제

1. 출입문 설정에서 자동문 사용 옵션을 **켬**으로 설정해도 화면상에 **끔**으로 표시되는 문제
