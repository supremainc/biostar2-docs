## 신규 기능 및 개선

---

1. 재실 인원 제한 구역 지원

    * 재실 인원 제한 구역: 한 공간에 많은 인원이 밀집하지 못하도록 입실 인원 수를 제한하고 재실 인원 현황을 모니터링할 수 있는 구역

1. 신규 장치 연결 지원

    * X-Station 2

1. FaceStation F2에 조명 밝기 조절 기능 명칭 변경 및 **사용 안 함** 옵션 추가

    * 기능 이름 변경: **주변 밝기**(**Ambient Brightness**)에서 **조명 밝기**(**Light Brightness**)로 변경

    * **사용 안 함** 옵션을 추가하여 장치의 조명을 끌 수 있도록 개선

    * **자동** 옵션 삭제
  
1. 비주얼 페이스 모바일 등록 및 QR/바코드 발급에 대한 이메일 설정 분리

1. Swagger 내 API 보강

    * 재실 인원 제한 구역 관련 신규 API 추가

    * 비주얼 페이스, 열화상 카메라 설정 및 온도 로그, CSV 내보내기 및 가져오기 관련 API 보강

    * X-Station 2 QR/바코드 관련 신규 API 추가

1. 일본어 리소스 파일 업데이트

## 버그 수정

---

1. 사용자에게 CSN 카드를 발급하면 개발자 도구의 Console에 **TypeError: Cannot read property 'replace' of undefined** 오류가 출력되는 문제 (발생 버전: v2.8.9)

1. 데이터 내보내기를 통해 BioStar 2의 데이터를 내보낸 장치에서 지문 인증에 실패하는 문제 (발생 버전: v2.8.10)

1. 데이터 파일이 호환되는 버전 펌웨어가 설치된 장치에서 이벤트 로그 가져오기를 실행하였음에도 호환 오류 메시지가 발생하며 데이터를 가져올 수 없는 문제 (발생 버전: v2.8.10)

1. 버전을 2.8.9 이상으로 업그레이드할 경우 2.6.4 이전 버전에서 생성한 사용자로 로그인 불가능한 문제 (발생 버전: v2.8.9)
