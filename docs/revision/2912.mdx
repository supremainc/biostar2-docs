## 신규 기능 및 개선

---

1. 신규 장치 연결 지원

   * BioStation 3 Max

   * XPass Q2

1. BioStar 2에서 장치의 이벤트 로그를 삭제할 수 있는 기능 추가

1. BioStar X 지원 펌웨어가 설치된 장치를 BioStar 2에서 사용할 때, 신규 이벤트 로그가 정상적으로 출력되도록 업데이트

1. <Cmd sid="setting.menu.remoteAccess.root" /> 기능 지원

1. 근태 관리 보안 취약점 개선

   * PDF 내보내기 시 발생하는 URL 노출 및 비인가 접근 취약점 개선

   * Minimist 라이브러리 보안 취약점에 따른 버전 업데이트

   * 스크립트 파일 내 하드코딩된 테스트 계정 정보 제거

   * DB 비밀번호 암호화 방식 강화

1. Open JDK 버전 업데이트

1. 시스템 보안 취약점 개선

   * 웹 서버 보안 헤더(HTTP Security Headers) 적용

   * 데이터베이스 접속 비밀번호의 암호화 저장 방식 강화

   * Java 서버 보안 취약점 관련 내부 라이브러리 업데이트

   * 디렉토리 연동 페이지의 비밀번호 보안 강화

   * 사용자 목록 페이지의 접근 권한 관련 취약점 개선

   * SQL Injection 보안 취약점 개선

1. 보안 강화를 위해 2.9.8 이상 버전으로 업그레이드 시, 불필요한 Redis가 설치되어 있을 경우 삭제를 권고하는 팝업 메시지 추가

1. 2.9.8 이하의 버전에서 업그레이드 시, 기존 사용자의 데이터를 보호하기 위해 첫 동기화 과정에서 연동 제외 설정을 자동으로 적용하도록 개선
