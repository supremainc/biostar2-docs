## 신규 기능 및 개선

1. 대시 보드에 비주얼 페이스 추가

1. 비주얼 페이스 모바일 등록 페이지 UI 개선

1. 비주얼 페이스 모바일 등록 요청 메일에 바닥글 추가

1. QR/바코드 발급 시 사용자 ID를 카드 ID로 사용할 수 있는 옵션 삭제

1. BioStar 2 QR로 QR/바코드 발급 시 이미지 크기를 변경하여 선명도 개선

1. Wiegand 카드 ID를 0으로 시작할 수 없는 제약 수정

    * 첫 번째 필드에 0을 입력할 수 있도록 허용

    * 네 개의 필드 데이터를 모두 0으로만 구성할 수 없도록 제약

1. CSV 불러오기에 실패한 사용자 목록 파일(User_Import_Error)에 실패 사유 열 추가

1. 사용자 정의 설치 시 AC, TA, VE 서비스가 동일한 데이터베이스 이름을 사용하지 못하도록 체크하는 로직 추가

1. BioStar 2 설치 시 포함된 크롬(Chrome)을 75버전으로 변경

1. USB Device Agent 보안 취약점 개선

1. 비디오 보안 취약점 개선

1. 데이터베이스 암복호화 시 TA, VE 데이터베이스에 동일한 테이블 이름이 존재할 경우 스키마명까지 확인하여 올바른 결과 값을 조회하도록 개선

1. 미할당 카드 삭제 옵션 추가

1. 퇴실 버튼 입력에 대한 릴레이 비활성 옵션 추가

    * 퇴실 버튼 입력 시 출입문 열림 요청 로그는 발생하지만 릴레이는 동작하지 않도록 설정할 수 있는 옵션 지원

1. 출입문 개방 사유 관련 로그 분리

    * 퇴실 버튼에 의한 출입문 개방 요청

    * 관리자에 의한 출입문 개방 요청

1. 스마트 카드 출력 바이트 순서 설정 추가

    * Wiegand 또는 OSDP로 출력되는 데이터 값에 대해 바이트 순서를 설정할 수 있도록 지원

1. setting.config 파일 초기화 로직 추가

1. 모바일 출입카드 발급 시 사용자의 전화번호가 Airfob Portal의 허용 범위를 벗어날 경우 포털로 전송하지 않도록 개선

1. 클라우드 이용 시 개인 정보 수집 및 이용 동의 확인 절차 추가

1. 출입문 이중 인증 설정에 **모든 장치** 옵션 추가

1. 재실 인원 제한 구역 모니터링 페이지를 모니터링 장치 화면 해상도에 맞게 출력하도록 개선

1. CoreStation의 슬레이브 장치로 X-Station 2를 연결할 수 있도록 지원

1. 얼굴 크리덴셜 미리보기 숨기기 옵션 추가

    * 개인 정보 보호를 위해 서버에 등록된 사용자의 얼굴 정보 미리보기 화면을 숨길 수 있도록 지원

1. 카드를 QR로 인증하기 옵션 추가

    * 장치에서 스캔한 QR 데이터가 사용자에게 발급한 카드 데이터와 동일할 경우의 인증 허용 여부를 선택할 수 있도록 지원

1. 액티브 디렉토리 계정으로 BioStar 2에 로그인할 수 있도록 지원

1. LCD 화면이 없는 장치로 사용자 데이터를 전송할 때 데이터에 사용자 이름을 포함하도록 개선

    * 적용 장치: CoreStation, BioEntry W2, BioEntry P2, XPass 2

1. 독일어, 프랑스어 리소스 파일 업데이트

## 버그 수정

1. 연결이 끊긴 슬레이브 장치를 선택해 상세 페이지에 진입 시 화면이 비정상적으로 출력되는 문제 (발생 버전: v2.8.6)

1. POST/api/users의 photo 값에 대한 설명이 binaryDatas로 잘못 기재된 문제 (발생 버전: v2.7.12)

1. 6개월 이상 장기 미출입 사용자를 조회할 때 해당 사용자의 마지막 인증 기록이 표시되지 않는 문제 (발생 버전: v2.5.0)

1. FaceStation 2에서 화면 보호기 사용 여부 선택 옵션을 펌웨어 1.5.0 버전 이상에서부터 지원함에도, 이하 버전 장치의 상세 페이지에도 해당 옵션이 표시되는 문제 (발생 버전: v2.8.10)

1. 조지아어로 설정한 권한 등급 이름이 감사 추적 메뉴에서 물음표로 표시되는 문제 (발생 버전: v2.7.6)

1. BioEntry W2 (BEW2-OHP) 장치의 CSN 카드, 스마트 카드, 모바일 카드 활성화 설정이 저장되지 않는 문제 (발생 버전: v2.7.6)

1. MS SQL Server 데이터베이스를 사용하는 환경에서 CoreStation의 슬레이브 장치로 추가된 Output Module(OM-120)을 삭제할 경우 장치 목록이 조회되지 않는 문제 (발생 버전: v2.8.12)

1. 복호화 환경에서 암호화 마이그레이션에 실패하는 문제 (발생 버전: v2.8.0)

1. 간헐적으로 비주얼 페이스 데이터가 다른 장치에 동기화되지 않는 문제 (발생 버전: v2.8.6)

1. 장치의 사용자 관리에서 사용자를 삭제할 경우 동기화 테이블의 다른 정보까지 삭제되어 동기화에 실패하는 문제 (발생 버전: v2.4.0)

1. **모든 장치(장치 사용자 변경 포함)** 사용자 자동 동기화 시 커스텀 사용자 필드를 사용 중인 사용자는 업데이트되지 않는 문제 (발생 버전: v2.8.0)

1. 경보 설정을 변경하면 감사 추적의 변경 대상이 **eventType.번호** 형식으로 출력되는 문제 (발생 버전: v2.8.9)

1. Airfob Portal 연결 시 알 수 없는 오류 메시지가 표시되며 연결에 실패하는 문제 (발생 버전: v2.7.12)

1. 전체 권한이 설정된 장치에 출입 등급을 설정할 수 있는 문제 (발생 버전: v2.4.0)

1. 릴레이를 설정하지 않으면 출입문을 생성할 수 없으나, 기존에 생성된 출입문은 릴레이 설정 없이 저장되는 문제 (발생 버전: v2.5.0)

1. 실시간 로그에 기록된 로그가 이벤트 로그에서는 조회되지 않는 문제 (발생 버전: v2.8.4)

1. 온도 측정 오류 팝업 메시지에 사용자 ID만 표시되고 이름은 출력되지 않는 문제 (발생 버전: v2.8.11)

1. 일광 절약 시간(DST) 설정이 이미지 로그 시간 정보에 반영되지 않는 문제 (발생 버전: v2.0.0)

1. 서브도메인을 사용할 경우 비디오 메뉴에 접속할 수 없는 문제 (발생 버전: v2.6.0)

1. 장치에서 추가된 사용자를 동기화할 경우 사용자 그룹의 카운트는 업데이트되지 않는 문제 (발생 버전: v2.0)

1. 아래와 같은 조건으로 안티패스백 구역을 구성한 장치를 소집 구역으로 중복 구성할 경우 정상 인증에 대해서도 안티패스백 위반 경보가 발생하는 문제

    * 안티패스백 구역의 입실 장치를 소집 구역의 퇴실 장치로 구성하고, 안티패스백 구역의 퇴실 장치를 소집 구역의 입실 장치로 설정

1. 장치 홈 화면 로고 및 슬라이드 쇼 설정 시 이미지가 장치 화면의 해상도에 맞춰 리사이징되지 않아 화면이 깨져 보이는 문제 (발생 버전: v2.6.0)

1. 개인 정보 DB 암호화를 사용할 때 액티브 디렉토리에서 사용자 정보를 편집한 뒤 동기화하면 변경 내용이 동기화되지 않는 문제 (발생 버전: v2.8.0)

1. 소집 구역 및 인터락 구역에 알람을 설정할 때 장치 수가 많을 경우 알람 추가 화면에 장치 목록 전체가 표시되는 문제 (발생 버전: v2.6.0)

    * 목록을 스크롤 형식으로 변경

1. 언어를 일본어로 설정했을 때 이벤트 로그를 CSV 파일로 내보내면 파일 내 텍스트가 영어로 출력되는 문제 (발생 버전: v2.8.9)

1. 장치 상세 페이지에 진입한 상태에서 장치 연결이 끊어졌을 때 **확인** 버튼을 클릭하면 상황과 무관한 오류 메시지가 표시되는 문제 (발생 버전: v2.7.10)

    * 장치가 연결되지 않았음을 알리는 메시지를 표시하도록 개선

1. 라이선스에서 지원하는 개수를 초과하여 출입문을 생성할 때 오류 메시지가 표시되지 않는 문제 (발생 버전: v2.8.3)

1. API를 이용해 사용자 정보를 업데이트할 때 기존과 동일한 이메일을 유지할 경우 **E-mail already exists** 메시지가 반환되는 문제 (발생 버전: v2.0.1)

1. **비주얼 페이스 추출 실패** 이벤트가 이벤트 로그 필터로 조회되지 않는 문제 (발생 버전: v2.8.10)

1. 사용자 정보의 부서, 직함 입력 시 특수 문자 사용을 지원하지 않음에도 API로는 특수 문자를 입력할 수 있는 문제 (발생 버전: v2.8.9)

1. 커스텀 사용자 필드가 존재할 경우 CSV 불러오기 후 사용자 상세 페이지에서 설정 변경 없이 확인을 클릭해도 동기화가 진행되는 문제 (발생 버전: v2.8.12)

1. QR/바코드를 발급한 사용자의 이름에 일부 기호(-)를 입력할 수 없는 문제 (발생 버전: v2.8.11)

1. 비주얼 페이스 모바일 등록 페이지에 불필요한 날짜가 표시되는 문제 (발생 버전: v2.8.11)

1. 액티브 디렉토리에서 동기화를 진행해도 마지막 동기화 시간, 마지막 업데이트 시간이 표시되지 않는 문제 (발생 버전: v2.7.11)

1. 모바일 출입카드가 설정된 상태에서 모바일 출입카드 설정 화면의 **연결** 버튼을 클릭하면 오류 메시지가 표시되는 문제 (발생 버전: v2.8.14)

1. 최선 버전 크롬(Chrome 98)을 사용하는 환경에서 USB 지문 등록용 장치(BioMini)로 지문을 등록할 때 지문 스캔 요청이 2회 동작하는 문제 (발생 버전: v2.0.0)

1. 근태 관리를 지원하지 않는 장치 상세 페이지에 근태 관리 관련 설정이 표시되는 문제 (발생 버전: v2.8.14)

1. 일부 근태 보고서를 2페이지 이상 PDF 내보내기 할 경우 기능이 정상적으로 동작하지 않는 문제 (발생 버전: v2.8.14)
