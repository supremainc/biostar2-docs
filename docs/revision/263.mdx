## 버그 수정

---

1. BioStar 2가 설치된 PC가 네트워크에 연결되지 않은 경우 설정의 비디오 메뉴가 화면에 출력되지 않는 문제

1. 하위 그룹을 가진 엘리베이터 그룹을 다른 엘리베이터 그룹의 하위로 이동하면 오류가 발생하며 BioStar 2가 정상적으로 동작하지 않는 문제

1. XPass D2(XPD2-GKDB) 장치 페이지의 화면/소리 항목에서 RTC와 DHCP 삭제

1. 이전 버전 펌웨어를 사용하는 XPass D2(XPD2-GKDB)를 BioStar 2에 연결해도 펌웨어 업그레이드 알림이 표시되지 않는 문제

1. 서버 매칭 기능을 활성화하였을 때 XPass D2(XPD2-GKDB)에서 키패드로 AoC ID를 입력해도 인증에 성공하는 문제

1. 엘리베이터를 추가하지 않은 상·하위 엘리베이터 그룹을 동시에 선택하여 이동할 경우 해당 그룹이 정상적으로 출력되지 않는 문제

1. 장치에서 날짜 형식을 변경한 뒤 BioStar 2 장치 페이지에서 적용 버튼을 누르면 날짜 형식이 **YYYY/MM/DD**로 바뀌는 문제

1. BioMini에 지문을 스캔하면 오류가 발생하며 지문 스캔이 정상적으로 진행되지 않는 문제

1. 사용자 이름을 아랍어로 설정하면 이름이 정상적으로 출력되지 않는 문제

1. 일반 사용자 계정으로 관리자 권한을 획득할 수 있는 보안 취약점 수정

1. BioStar 2가 설치된 PC에서 Java 1.8.0 이상을 사용할 경우 BioStar 2 버전 업그레이드에 실패하는 문제

1. 근태 보고서에 사용자의 일부 근태 기록이 누락되는 문제
