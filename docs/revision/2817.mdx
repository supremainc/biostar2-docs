## 신규 기능 및 개선

---

1. 근태 관리 보안 취약점 개선

    * 서버 설정 시 지원하지 않는 Cipher suite 사용을 제한

    * DB 드라이버 최신 버전 적용

    * jQuery 최신 버전 적용

    * 웹 소켓을 통한 근태 관리 API 사용 시 보안 확인 절차 추가

1. 일정 기간이 지난 근태 기록 삭제 기능 지원

1. 시간 기록을 생성하는 API에 장치 이름(devnm)을 사용자 정의로 지정할 수 있도록 개선
