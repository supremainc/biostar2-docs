### 릴리즈

2019-04-11

## 신규 기능 및 개선

1. 액티브 디렉토리 메뉴 추가

    * Windows Active Directory 사용자 데이터를 BioStar 2에서 사용할 수 있도록 지원

1. 사용자, 장치, 출입문 메뉴에서 목록 정렬 지원

    * 사용자: 그룹, 상태 열

    * 장치: RS-485 열

    * 출입문: 그룹 열

1. 지원하지 않는 AoC 관련 이벤트를 장치, 비디오, 설정의 작업 조건 및 동작에서 제거

1. setting.conf에서 9000번 포트를 변경할 수 있도록 개선

1. FaceStation 2 슬레이브 장치 설정에 주변 밝기 옵션 추가

## 버그 수정

1. 장치 메뉴에서 스마트 카드 레이아웃을 설정할 때 iCLASS Seos와 DESFire Advanced 카드 포맷이 적용되지 않는 문제

1. 경비 구역에서 장치를 경비 개시/해제 장치로 설정한 뒤, 해당 장치로 구성된 출입문을 제외할 경우 **Failed to parse JSON(700)** 오류가 발생하는 문제

1. 사용자 그룹을 추가하거나 편집할 때 사용자 그룹 정보가 정상적으로 생성되지 않은 경우, 해당 그룹을 선택하면 사용자 목록이 출력되지 않는 문제

1. 커스텀 권한을 249개 이상 추가한 경우에도 화면상에 249개까지만 표시되는 문제
