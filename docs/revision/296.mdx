### 릴리즈

2024-06-12

## 신규 기능 및 개선

1. 신규 장치 연결 지원

    * XPass 2 (XP2-MAPB)

2. **인텔리전트 슬레이브**를 사용할 경우 타사 컨트롤러의 인증 성공/실패 결과를 장치 화면에 표시하는 **컨트롤러 결과 표시** 기능 지원

3. 장치를 타사 컨트롤러와 Wiegand로 연결했을 때 컨트롤러로부터 반복적으로 입력되는 인증에 대한 신호를 무시하는 **반복 신호 무시 시간** 기능 지원

4. **IP 인터폰**, **RTSP** 사용 시 장치에서 송출되는 영상의 해상도를 선택할 수 있는 기능 지원

    * **IP 인터폰 비디오 해상도**

    * **RTSP 비디오 해상도**

5. 장치의 인증 결과 화면에서 사용자 ID, 이름이 표시되는 방법을 선택할 수 있는 인증 결과 표시 옵션 기능 지원

    * **사용자 ID 표시**

    * **사용자 이름 표시**

6. 장치에서 **IP 인터폰**의 SIP 서버 설정 시 SIP 전송 방법을 선택할 수 있는 **SIP 서버 전송** 기능 지원

7. 장치 **일괄 편집**에 **인증 모드** 편집 기능 지원

8. FaceStation F2(펌웨어 v2.1.4 이상) - 장치가 설치된 조도 환경에 따른 깜빡임 발생 방지를 위한 **주파수** 조정 옵션 추가

9. **커스텀 스마트 카드 레이아웃** 설정에서 FeliCa 설정 지원

10. X-Station 2 **얼굴 검출** 기능 지원 중단

11. 보안 취약점 개선

    * XSS 보안 취약점 개선

    * 웹 소켓 관련 보안 취약점 개선

    * CSRF 보안 취약점 개선

    * 7zip 버전 업그레이드

    * SSL/TLS Diffie-Hellman Modulus의 보안 취약점 개선

    * Spring Framework 버전 업그레이드

    * HTTP Content-Security-Policy 취약점 개선

    * 모든 입력창에서 자동완성 기능 삭제

    * SQL Injection 보안 취약점 개선

    * BioStar 2 인증서 보안 개선

    * **액티브 디렉토리** 계정 설정 보안 취약점 개선

12. 근태 관리 보안 취약점 개선

    * Access-Control-Allow-Origin 보안 취약점 개선

    * 근태 관리 API 문서에 X-Frame-Options 추가

13. **근태 관리** > **근무 스케줄** 메뉴의 사용자 조회 화면 성능 개선

14. **근태 관리** > **근태 기록** 메뉴의 근태 기록 상세보기 화면(캘린더, 목록)에 새로고침 버튼 추가

15. 일광 절약 시간(DST)을 사용하지 않는 국가에서 근태 기록의 시간 표시 방법 개선

16. 오래된 근태기록 데이터가 많을 경우 최신 데이터의 동기화 속도 개선

17. **경보 목록**의 **사용자** 열에 사용자 이름과 사용자 ID가 모두 표시되도록 개선

18. **경보 목록**의 **날짜** 열에 시간이 초 단위까지 표시되도록 개선

19. **사용자 자동 동기화** - **특정 장치(해당하는 출입 그룹에 속한 장치만 동기화)** 모드 사용 시 성능 개선

    * **구역** 추가, 수정, 삭제 시 속도 개선

    * **출입 등급** 변경 시 속도 개선

20. **근태 관리**의 **사용자 그룹** 및 **장치 그룹** 동기화 개선

21. **사용자 ID 종류**가 **숫자**로 설정된 상태에서 파일명이 0으로 시작하는 비주얼페이스 파일 업로드 시 표시되는 오류 팝업 메시지 수정

22. **사용자 ID 종류**를 **영숫자**로 사용할 때 중복 사용자 ID에 대한 체크 로직 개선

23. C에서 Java로 내부 API를 호출할 때 고정 IP를 사용하도록 개선

24. 특정 장치의 인증모드 최대 갯수 증가

25. MS SQL 서버 데이터베이스를 사용하는 환경을 위한 신규 인덱스 추가

26. CSN 모바일 카드 발급 시 카드 아이디에 입력 가능한 최대 글자 수 조정

27. 암호화 키 저장 경로를 사용자가 직접 설정할 때 경로에 특수 문자가 있는 경우 팝업 메시지를 출력하도록 개선

28. **작업 조건 및 동작** 설정 화면의 장치 목록에 Wiegand 리더가 나타나도록 개선

29. BioStar 2 서버를 시작하기 전에 서버 시작 가능 여부를 체크하여 문제가 있는 경우 팝업 메시지를 출력하도록 개선

30. 이벤트 로그를 **CSV 내보내기** 또는 **출력**했을 때 사용자 관련 이벤트 로그가 장치 또는 서버 중 어디에서 발생했는지 구분하여 표시하도록 개선

31. **비밀번호 최대 실패 허용 횟수** 초과 발생 시 이벤트 로그, 실시간 로그 및 경보에 기록되도록 개선

32. 최대 100명의 사용자 정보를 수정할 수 있는 API 추가

33. **비밀번호 최대 실패 허용 횟수** 제한 옵션이 기본값으로 적용되도록 개선

34. 출입 그룹 생성 시 필요한 조건 완화 및 설정되지 않은 항목에 대한 알림 팝업 메시지 추가

35. 로그인 API 개선

36. 모바일 카드 ID 검사 로직 개선

37. BioStar 2 Setting 화면 UI 개선

38. BioStar 2 Setting에서 서비스들의 포트를 사용 중일 때 상태창에 표시되도록 개선

39. 스페인어 리소스 업데이트

## 버그 수정

1. 특정 SMTP 환경에서 이메일 발송이 실패하는 문제 (발생 버전: v2.6.0)

2. 특정 장치에서 Global APB와 이중 인증모드를 사용할 경우 첫번째 사용자의 인증이 되지않는 문제 (발생 버전: v2.9.0)

3. **화재 경보** 구역 목록에서 **활성/비활성** 열이 오름차순, 내림차순으로 제대로 정렬되지 않는 문제 (발생 버전: v2.4.0)

4. 방문자 라이선스가 없는 상태에서 사용자의 **권한 등급**에 **방문자 관리자**가 표시되는 문제 (발생 버전: v2.7.7)

5. 일부 항목의 아랍어 표시 오류 (발생 버전: v2.8.9)

6. 연결이 해제된 장치에 비주얼페이스 템플릿을 사용하여 스마트카드를 발급할 경우 카드가 발급되는 문제 (발생 버전: v2.8.2)

7. 특정 환경에서 런처 서비스(biostar-server.exe)가 종료되는 문제 (발생 버전: v2.0.0)

8. **출입 통제** 메뉴의 **상태**에서 **엘리베이터 권한 상태 - 층별**, **엘리베이터 권한 상태 - 사용자별** 목록을 **CSV 내보내기** 했을 때 사용자 ID와 이름이 암호화되어 출력되는 문제 (발생 버전: v2.8.0)

9. BioStar 2 서비스 재시작 시 **서버 매칭**의 얼굴 **보안 레벨**과 **이미지 로그**의 **이미지 로그 없는 경우 사용자 프로필 이미지 표시** 설정이 초기화되는 문제 (발생 버전: v2.2.1)

10. 특정 장치를 잠금 상태로 변경한 후 소리가 출력되지 않는 문제 (발생 버전: v2.9.1)

11. 특정 그룹이나 등급에 속한 장치가 **전체 권한** 설정이 가능한 문제 (발생 버전: v2.8.16)

12. 장치 그룹에 포함된 장치를 전체 선택하고 **데이터 삭제 후 장치 동기화**를 진행하면 모든 장치가 동기화 되는 문제 (발생 버전: v2.0.0)

13. 장치 **일괄 편집**으로 장치의 설정 변경 시 **카드를 QR로 인증하기** 기능이 활성화되는 문제 (발생 버전: v2.8.16)

14. 사용자 메뉴 권한이 없는 커스텀 권한을 가진 계정으로 로그인 및 로그아웃한 후 전체 관리자 계정으로 로그인하여 **사용자** 메뉴 진입 시 **새 사용자 추가** 화면이 출력되는 문제 (발생 버전: v2.7.2)

15. **사용자 자동 동기화** - **특정 장치(해당하는 출입 그룹에 속한 장치만 동기화)** 모드 사용 시 방문자의 크리덴셜을 수정하면 장치와 동기화되지 않는 문제 (발생 버전: v2.8.14)

16. 모바일 출입 카드에 등록된 장치가 삭제되지 않는 문제 (발생 버전: v2.8.7)

17. 출입문 목록을 인쇄했을 때 출입문의 **상태**가 **정상**으로만 표시되는 문제 (발생 버전: v2.4.0)

18. 크리덴셜 등록 시 선택했던 장치가 삭제된 후에도 **사용자** > **크리덴셜** 창의 장치 목록에 표시되는 문제 (발생 버전: v2.0.0)

19. 인증용으로 사용할 수 없는 이미지가 정상적으로 등록되는 문제 (발생 버전: v2.9.0)

20. 서력을 사용하지 않는 운영체제에서 이벤트 로그 삭제 시 오류가 발생하던 문제 (발생 버전: v2.6.0)

21. 특정 구역의 **열 설정** 시 **열 목록**이 비정상적으로 표시되던 문제 (발생 버전: v2.5.0)

22. 장치 ID가 일정 글자 수를 넘을 때 장치 그룹이 비정상적으로 표시되는 문제 (발생 버전: v2.6.3)

23. **재실 인원 제한**을 지원하는 장치가 경비 상태일 때 **데이터 삭제 후 장치 동기화**를 진행하면 동기화에 성공하는 문제 (발생 버전: v2.8.11)

24. 로그인 중인 사용자의 이름 변경 시 우측 상단의 사용자 이름이 갱신되지 않는 문제 (발생 버전: v2.0.0)

25. 사용자가 없는 사용자 그룹에서 **비주얼 페이스 모바일 등록 링크 전송** 실행 시 오류 팝업 메시지가 출력되지 않는 문제 (발생 버전: v2.8.6)

26. **사용자 그룹**이 8단계로 구성되어 있는 CSV 파일을 불러올 경우 **사용자 그룹**이 정상적으로 추가되지 않는 문제 (발생 버전: v2.4.0)

27. **CSV 불러오기** 시 사용자 유효기간이 범위를 벗어나는 경우 나타나는 오류 메시지 수정 (발생 버전: v2.8.16)

28. **스케줄 잠금**, **스케줄 개방** 구역 설정 시 **출입문** 항목에 선택된 출입문과 RS-485 네트워크가 다른 출입문이 검색되는 문제 (발생 버전: v2.4.0)

29. 로그인 후 **설정** > **작업 조건 및 동작** 설정 화면에 진입 시 **동작** - **신호 출력** 부분의 목록이 나타나지않는 문제 (발생 버전: v2.0.0)

30. 모든 장치가 서버에 연결되어있지 않은 상태에서 **사용자 ID 종류** 변경이 가능한 문제 (발생 버전: v2.4.0)

31. **사용자** 메뉴에서 **사용자 그룹**을 기준으로 열을 정렬했을 때 PDF가 제대로 출력되지 않는 문제 (발생 버전: v2.7.8)

32. 암호화 사용 환경에서 사용자 ID에 16자 이상을 입력하고 복호화한 경우 사용자 ID 정렬에 오류가 발생하는 문제 (발생 버전: v2.8.9)

33. BioStation 2a(BS2A-ODPB) 모델의 장치 상세 페이지에서 지원하지 않는 메뉴가 나타나는 문제 (발생 버전: v2.9.4)

34. **소집 구역**에 설정된 **출입 그룹**의 사용자 정보가 변경된 경우 **소집 구역**의 사용자 정보에 동기화 되지 않는 문제 (발생 버전: v2.6.0)

35. 특정 사용자 그룹의 사용자를 전체 선택한 후 **비주얼 페이스 모바일 등록 링크 전송**을 시도할 경우 그룹에 포함되어 있지 않은 모든 사용자에게도 이메일이 전송되는 문제 (발생 버전: v2.9.0)

36. 등록된 장치에 API를 사용하여 장치 타입을 변경할 수 있는 문제 (발생 버전: v2.5.0)

37. User API를 사용하여 사용자를 생성 또는 수정할 때 일부 크리덴셜의 값이 없는 상태에서 API를 호출할 경우 로그아웃되는 문제 (발생 버전: v2.0.0)

38. **이벤트 로그**에서 **데이터 파일 가져오기** 시 간헐적으로 로그아웃 되는 문제 (발생 버전: v2.9.1)

39. **사용자** 메뉴에서 **데이터 파일 가져오기** 시 특정 조건에서 오류 팝업이 발생하지 않는 문제 (발생 버전: v2.9.3)

40. 상태가 **알 수 없음**인 **커스텀 스마트 카드 레이아웃**이 장치와 동기화되는 문제 (발생 버전: v2.9.4)

41. **커스텀 스마트 카드**가 설정된 장치의 **커스텀 스마트 카드 레이아웃**을 **없음**으로 변경하면 장치와 기본값으로 동기화되는 문제 (발생 버전: v2.9.4)

42. 일부 장치 그룹에만 권한이 있는 사용자가 카드 등록 시 권한이 없는 장치들도 목록에 나타나는 문제 (발생 버전: v2.8.14)

43. **출입 통제** 메뉴 진입 시 간헐적으로 로그아웃되는 문제 (발생 버전: v2.2.0)

44. **근무 스케줄**에서 사용자가 등록되지 않은 스케줄의 +, -버튼 클릭 시 오류 팝업 메시지가 출력되는 문제 (발생 버전: v2.8.1)
