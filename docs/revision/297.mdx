## 신규 기능 및 개선

---

1. 신규 장치 연결 지원

    * BioEntry W3

1. 기존의 ID + 비밀번호 조합에 지문 스캐너를 이용한 지문 인증 단계를 추가하여 사용자 계정의 보안을 향상시킬 수 있는 **다중 인증 로그인** 기능 추가

    * 지원되는 지문 스캐너

      * BioMini

      * BioMini Plus 2

1. 사용자 동기화 시 비주얼 페이스에서 실제 이미지를 제외한 템플릿만 전송하여 동기화 소요 시간이 단축되도록 동기화 프로토콜 개선

1. **전문가 설정** 메뉴 추가

    * **비주얼 페이스 이미지 저장하기** 기능 추가

      * 비주얼 페이스에 실제 이미지와 템플릿을 모두 저장하도록 하거나 실제 이미지는 저장하지 않고 템플릿만 저장하도록 설정할 수 있는 기능 지원

1. 웹 서버 cipher suite 관련 보안 취약점 개선

1. **커스텀 사용자 필드**에 대한 XSS 보안 취약점 개선

1. **장치 라이선스**에서 **미지원** 장치에 대한 안내 툴팁 추가

1. 다국어 리소스 업데이트

    * 프랑스어, 이탈리아어, 일본어, 루마니아어

## 버그 수정

---

<BugLists>

1. 데이터베이스에 올바르지 않은 문자열이 존재하는 경우 서버가 종료되는 문제 <Badge>2.5.0</Badge>

1. **특정 장치 사용자 자동 동기화** 모드에서 동기화에 실패한 사용자를 연결이 끊어진 장치에 재시도하는 문제 <Badge>2.8.15</Badge>

1. 글로벌 프로토콜 사용 시 global check violation invalid packet에 대한 오류 수정 <Badge>2.8.11</Badge>

1. **지문 LFD** 기능을 지원하는 특정 장치의 상세 페이지에 **지문 LFD** 기능 UI가 보이지않는 문제 <Badge>2.9.0</Badge>

1. FaceStation F2와 FaceStation 2를 함께 사용하는 환경에서 **사용자 자동 동기화** - **모든 장치(장치 사용자 변경 포함)** 모드를 사용 중일 때 BioStar 2 서버에 얼굴이 등록되어 있는 사용자에게 FaceStation F2에서 비주얼 페이스를 추가하는 경우 BioStar 2 서버에 비주얼 페이스가 추가되지 않는 문제 <Badge>2.8.6</Badge>

1. 연결 해제된 장치에 비주얼 페이스 템플릿을 사용하여 스마트 카드를 발급할 경우 **미할당 카드** 및 **사용 정지된 카드**에 추가되는 문제 <Badge>2.9.6</Badge>

1. 장치의 이름을 다른 장치에서 이미 사용 중인 중복된 이름으로 변경했을 때 간헐적으로 오류 팝업 메시지가 나타나지 않는 문제 <Badge>2.0.0</Badge>

1. 웹캠을 사용하여 프로필 사진 등록 시 GPU 사용량이 비정상적으로 상승한 후 감소하지 않는 문제 <Badge>2.9.5</Badge>

1. **사용자** 메뉴에서 목록에 사용자가 표시되지 않는 문제 <Badge>2.9.5</Badge>

1. **커스텀 리포트** 생성 시 리포트 결과의 이벤트가 상당 수 이상일 경우 전체 데이터가 생성되지 않는 문제 <Badge>2.9.3</Badge>

1. MS SQL Server 데이터베이스를 사용하는 환경에서 시스템 백업 및 복원에 실패하는 문제 <Badge>2.9.3</Badge>

1. 크롬(Chrome)의 **Keyboard focusable scrollers** 기능이 활성화되어 있는 경우, 스크롤이 있는 드롭다운 목록에서 원하는 옵션을 제대로 선택할 수 없는 문제 <Badge>2.0.0</Badge>

1. MS SQL Windows 인증으로 BioStar 2를 설치한 환경에서 **BioStar 2 Restore**를 통한 데이터베이스 복원이 실패하는 문제 <Badge>2.9.3</Badge>

</BugLists>
