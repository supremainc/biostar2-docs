이 문서에서는 BioStar 2.9.8 버전에서 업데이트된 기능들을 소개하고 사용하기 쉽도록 안내합니다.

## 마스크를 착용한 경우 인증을 거부하도록 설정하기

사용자가 마스크를 착용하고 있는 경우 얼굴 인증을 거부하도록 설정하여 얼굴 인증의 정확도를 높일 수 있습니다.

1. <Cmd sid='common.gnb.device' /> 메뉴를 클릭하세요.

1. 장치 목록에서 설정하려는 장치를 클릭하세요.

:::info

<Cmd>마스크 착용 시 인증 거부</Cmd> 옵션은 BioStation 3 펌웨어 1.3.1버전 이상에서만 사용할 수 있습니다.

:::

3. **마스크** 설정에서 **마스크 검출** 기능을 **사용** (**마스크 착용 시 인증 거부**)로 설정하세요.

<Image src='/img/ko/device_mask.png' alt='마스크' />

<center>위 이미지는 예시 화면입니다. 실제 화면과 다를 수 있습니다.</center>

:::info

**마스크 검출** 기능을 **사용** (**마스크 착용 시 인증 거부**)로 설정한 경우 **마스크 확인 방식**이 **인증 전 확인**으로 고정되며 변경할 수 없습니다.

:::

4. 설정을 저장하려면 **확인**을 클릭하세요.
